2016/1002Sun井上小百合ヽ(｡･ω･｡)コウモリかモグラ

 
 
 
 
◎ 「別冊カドカワ」10/3 発売
 
好きな曲についてお話しました。
悩んで悩んで選び出したお気に入りの5曲！！
 
何を選んだかはお楽しみにっ( *ˆoˆ* )
 
 
それから、舞台についてもお話させて
頂いてます♩ ぜひぜひチェックしてね！
 
 
 
◎ 「乃木坂工事中」10/2 放送 ←今日！
 
いよいよ夏休みの課題発表です。
みんな、全国ツアーの合間をぬって頑張ってたのでぜひ見てほしいなっ
 
私も空中ブランコに挑戦したよ(﹡ˆᴗˆ﹡)
 
 
ー
 
 
ここ３日間、京都でお仕事してました♩♩
 
まずは、セブンイレブン限定ミニライブ
来て下さった皆さん、有難うございました
 
 
 
それから、
15th最後の全国握手会
 
ペアはかりんちゃんと！
 
かりんちゃんも、かりんちゃん推しの皆さんも
有難うございました！
 
 
 
ミニライブで、「行くあてのない僕たち」を
披露するのもこれが最後でした！
 
次できるとしたらもうバースデーライブ
くらいになっちゃうのかな(◎o◎)
 
 
この曲を歌えて、本当に本当に有難かったです。
これからも大切な一曲です。
 
 
 
 
そして、本日は個別握手会でした**
 
 
◎私服
：earth music&ecology
：DHOLIC
：DHOLIC
：adidas
：zoff CLASSIC ボストン
時計：G-SHOCK
 
 
髪は、編み込みしてました。
 
 
 
レーンには、ファンのみんなが作ってくれた
ユニットのポスターが沢山！！！！( *ˆoˆ* )
 
 
あったかいね  嬉しくてほっこり(•ᵕᴗᵕ•)
 
なんか映画のポスターみたいだ！（笑）（笑）
こんなかっこよく作ってくれて有難う
 
 
 
実は今日は、リクエストに応えて
例の衣装も着たんだけど、恥ずかしいので
写メなしで悪しからず...笑
 
来てくれた方だけのあれってことで！
 
ハロウィンやクリスマスに近い日程のときも、
終日それなりの衣装着ようと思ってます。
新年一発目もなんかやろうかなあ(+o+)
恥ずかしいので、来てくれた方のあれってやつで！笑
 
 
ー
 
 
 
すごいことに気づいてしまったぞ( *o* )
 
なんと...、今...、10月らしい！！！！！
 
ひゃーーーー！(ホラー的悲鳴)
 
 
知ってた？？ ( °_° )ふぇ～
 
 
風が冷たくなってきたね、
 
 
あったかくして寝てね！お休み！
 
 
 
 
 
                  さゆヽ(｡･ω･｡)正義
