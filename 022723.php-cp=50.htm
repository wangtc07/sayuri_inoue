2015/0207Sat井上小百合ヽ(｡･ω･｡)嬉しいお知ら せ

 
 
 
こんばそわヽ(^ω^)ﾉ
 
 
こないだは、節分でしたね！
 
季  節  の  分  かれ目○゜
 
早くあったかくなってほしいなぁ。
寒いの苦手だから( > <  )   ←切実
 
ギュッ
 
 
こういう行事のとき、いつも家族を思い出します。
 
学校から帰ってくると、
お家の玄関にイワシの頭が挿してあって
 
あぁ今日は節分だあ 豆まきだあ＾＾！
 
って、ウキウキしながら
「ただいまー」ってしてたのが、懐かしいです  ＊＊
 
 
家族みんなで、鬼はー外ー！福は内ー！
って家中を回って豆まきしたのも
懐かしい(*^_^*)
 
毎年 お父さんが鬼のお面をして、
「うわーこわいぞー」って、私とお兄ちゃんと妹を驚かすの！
 
楽しかったなー(o^^o)
 
 
 
今年の節分は撮影で忙しかったので、
次の日に、小さい手巻きを買ってきて
一人で黙々と食べました（笑）
 
 
みなさんは、どんな節分を過ごしましたかー？  ＋＋
 
 
  ー
 
 
質問にこたえるよ(ミ＾o＾ミ)
 
 
▼  おなまえ：はや●777● さん
     ━━━━━━━━━━━━━━
 
＊  白以外に好きな色は？笑
 
 
◎   水色が好きです(*^_^*)
 
成人式のときも、当日は白の振り袖を着たけど、
前撮りのときは、水色にしました。
 
 
 
 
 
  ー
 
 
みなさんにお知らせです！！
 
やっとやっと、わたくしめ、のぎ天のサークルに入りましたー！⊂( ˆoˆ )⊃
 
サークル活動第一回目に、他のお仕事で参加できなかったため、どこにも所属出来てなかったんですが
ついに料理部に入ったよー！嬉しいです！(*^_^*)
 
今回は、もうすぐバレンタインということでチョコを皆で作りました。
 
告白とかも...(///)照
 
良かったら見てください！
告白シーンはとばしてください！！笑
 
 
 
それから、舞台【第二章】學蘭歌劇『帝一の國』－決戦のマイムマイム－に
樋口日奈ちゃんと、Wキャストで出演させて頂くことが決定しました！
 
本当に嬉しい！有難いです( >_< )
 
 
早速色んなメンバーが「絶対観に行くね！」って言ってきてくれて、
本当にうれしい限りです。
 
 
この作品と、あとは
単純にお芝居がすごく好きなので、
またひとつ表現者として成長できたらいいなあ、
と思います。
 
原作の美美子ちゃんを壊さないように、
だけどしっかり、井上小百合の演技を見せれるように。
 
 
 
あと、明日、 23:30〜 NHK BSプレミアム「乃木坂46 SHOW!」
 
見てくださーい(*^_^*)
 
恥ずかしい...///
 
 
  ー
 
みなみお誕生日おめでとう！
素敵な一年になりますように(　ˆᴘˆ　)♪
 
 
 
せいたんの卒業が発表されました。
 
多分、ものすごく寂しくなるんだろうけど、
彼女の選んだ道を応援したいです。
 
 
私も、今ここで出来ることを
精一杯頑張らなきゃ！
 
 
  ー
 
 
さて、明日は、
「透明な色」スペシャル個別握手会＠東京ビッグサイト
ですね( ^ω^ )
 
 
アルバムの握手会ということで、
好きな曲の衣装や制服を自分で選んで着ます！
私が何を着るのかお楽しみにー ♪♪
 
 
あーでも私が咄嗟の衣装とか着て握手したら、
 
あの機敏にパフォーマンスしてるときとのギャップに、
びっくりしちゃうよなあ...(  ´^_^` )
 
それも面白いかなー？？笑
 
 
 
日向ぼっこ、好き(　ˆᴘˆ　)
ぽわぽわ
 
 
 
 
 
                           さゆヽ(｡･ω･｡)正義
