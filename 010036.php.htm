2013/0126Sat井上小百合さゆにゃん戦隊。＋＋

 
 
たっだいまー！( *^ω^ * )
 
もおずっと前に帰ってきてたけど
 
 
 
制服、さむいっ（ ; ; ）震っ
 
あっためて？( *pωq* ) ♡///
 
 
みなさんは風邪引いてないかなあ？
コメントで結構インフルエンザ流行ってるって
聞くから心配ですっ(つω<。｀)
 
 
 
さてさて、今日から
テスト 始まりました！
 
 
今日の科目は、日本史・倫理
体育・家庭総合の４教科でした！
 
 
出来ばえは...
 
 
ばっちしd|｀･∀･´d|ﾃﾞｽ
 
 
 
みんながパワー送ってくれたから
頑張れたよお～っ(´◡`๑) 
ありがとう
 
じゃあ私からも、
ありがとパワーーーうぇいおえぁ！！！！！ヽ(｡ ^ω^ ｡)ﾉﾞ ズガゴゴゴゴゴッ
 
 
／ すごーす！すごーす^o^！ ＼
 
 
 
明日もテストなので、
頑張りますっ (丿^ω^ヽ) ☆
 
今日もPC使ってお勉強なの♪
あ、前に、小学生のときに自分の
PCを買ってもらって、使いすぎて親に
ネット回線切られたって言ってたのが
昨日の相方くんです(´◡`๑) 笑
 
でも、お勉強のためだから！って
お願いして、中学のとき
復活させてもらったんです(๑òωó๑)
小学生のときから一緒にいる、
大切な私のパソコンなのっ
 
だから、いい子にちゃんとこのコで
お勉強してあげなきゃ꒰๑͒ • •`๑͒꒱
 
 
 
 
 
○  コーヒー好きなの？？？
コーヒーはブラックで飲んでる?
 
＊  まったくです（ ;  ; ）
  全然飲めないんですけど、、
  眠気覚ましにと思って...
  ブラックってにがいやつですか？？
 
  今日はちゃんとメガシャキ
  買ってきました！(｡> <｡)
 
 
○  さゆにゃんの好きな偉人教えてー？ww
 
＊  フローレンス・ナイチンゲールさんです！
 
  小さい頃、ナイチンゲールさんの
  本を読んで、すごく好きになって、
  こんな人になりたいって
  尊敬して憧れていました。
  将来は、看護師さんになりたいって
  言ってました、、(´^ω^｀; )
 
  小学生のときの文集にも、
  将来の夢は看護師さんになること
  って作文を書いてました(*∩ω∩*)
 
  誰かのためになりたいって、
  その気持ちは今も変わりません。
 
 
○   さゆにゃんって特別に何か風邪予防してますか?
 
＊   手洗い、うがい、消毒、マスク着用...
  あとは、マフラーや手袋をつけて
  身体を冷やさないようにしたり
  しています(*^ω^*)
 
   とくに私はのどが弱いので、
   暖房とかにも気を使ってます(｡> <｡)
  
 
   みなさんも風邪、気をつけてね！
   でも、もし風邪ひいちゃったら、
   いつでもここにきてね(*^_^*)♡
 
 さゆにゃんが元気あげるっ(๑>ω<๑)!!
 
 
 
さゆにゃんヽ(｡･ω･｡)
