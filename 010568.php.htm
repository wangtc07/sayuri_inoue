2013/0224Sun井上小百合ヽ(｡･ω･｡) クレヨン。

 
 
 
 
 
こんばんわ○°
 
 
お仕事の方、学校の方、
お疲れさまですっ( *^_^* )
 
お休みだった方は素敵な休日は過ごせたかな？
 
今日もブログ読んでくれて有難う＊
 
 
キリッ
 
生写真のスーツ姿です！
たまにはこんな格好もどうかな(*^_^*)？
 
教育実習生みたいだっ！
みなさーん！席についてくださーい！
 
 
 
今日は、お仕事でちょっと
遠くに行ってきました！
 
さむいっ！とにかくさむかったですっ！
ｷﾋﾞｨｨィイイイ！( ^o^ )
 
でも、人の優しさが温かかったです！
 
 
いつかプライベートでどこか遠くへ
ひとり旅したいなー！と思っているのです！
 
色んな人や色んな発見に出会いたい！
優しさに触れて強くなりたい！
 
 
 
 
 
タイミングを逃してしまってずっとおめでとうが書けてなかった、
１月２月お誕生日のメンバーの
お祝いブログを、乃木坂ハッピーバースデーの流れで書こうと思います( *^o^* )
 
 
まずは、
 
(  1．22  *  )
 
いくちゃん！
 
歌もダンスも演技もピアノもお仕事も
なんでも、器用にこなしちゃいます。
でも、完璧に見えるけど、多分そこにはものすごい陰の努力があるんだろうなあと思います。
話すと天然で面白くて料理はものすごいけど(o^^o)笑
そんなギャップも好きです♡
 
 
れなりん！
 
れなりんは、本当に真面目で頑張り屋さん。
頭も良くて、学業を頑張りながらお仕事もしっかりこなしててすごいなあって思います。
でも、普段はそんなそぶり見せないくらい、すっごいお茶目なんですよ＾＾
ほんとは両立でかなりハードだろうに...大変なときはいつでも頼っていいんだからね(o^^o)？
無理しすぎないでね(｡> <｡)＊
 
 
(  1．4  *  )
 
みさみさ ！
 
目がすごくきれいで、本当にビー玉みたいです！(#^o^#)きゅん
あと、歌が上手です！
負けず嫌いで頑張り屋さんで、なんでも一生懸命やりこなす人です。
よくレッスンのあと一緒に残って自主練したりします(丿^ω^ヽ)
頼れるお姉ちゃんみたいな存在っ！
 
 
(   2．6   *  )
 
みなみ！
 
いつも、カバンの中にたくさんお菓子やパンが入っています（笑）
 
ふわふわしてるので、みなみとわたしが話をすると会話が噛み合いません∧( 'Θ' )∧
でもなんかイエーイって感じです（ミ＾o＾ミ）
たまにイエーイだけで会話するときがあります（笑）
そんなみなみがかわいいです♡
 
 
(   1．31   *   )
 
ひなちま！
 
わたしの推しです！
うおおおひなちまかわいいよぉぉおお
本当にかわいい(* ^p^ *)！
でも、ダンスがめっちゃうまくて、ダンスしてるときは本当にすごい。演技とかも上手いと思います。
あと、ものすごく礼儀正しくって育ちのいい子なんだなあということが伝わってくるしかわいいもうかわいいしか言ってないやかわいい
 
 
(   2．8   *   )
 
かずみさん！
 
オーディションで一緒だったけど、そのときからめっちゃ面白くて
審査員を笑わせてたすごい子！ってのが第一印象(o^^o)◎
いつもみんなを笑わせてくれるかずみんだけど、実は、色々考えてたりしてて、めっちゃ優しくてすごくいい子なんです。
かずみんの笑顔にいつも元気をもらってます(* ^o^ *)有難う
 
 
(   2．17   *   )
 
ちはるん！
ときどき、どっちが年上かわからなくなるくらい大人っぽくて美しいです！
でも、楽屋ではいっつもうるさいくらい面白いの（ミ＾p＾ミ）
あと、めっちゃ目を近づけてくる！（笑）
ちはるんといると笑いが止まらないですっ！
これからも、みんなを笑顔にさせてください(｡> <｡)♡
 
 
(   2．20  *    )
 
 
まりか！
オーディションで初めて知り合ったのがまりかで、一緒に受かろうねって言ってたのを覚えてる。
受かったとき、2人で泣きながら抱き合って喜んだね(｡vωv｡｀)
ダンスがうまくて、踊ってるときのまりかがすごくスキ！！！
あと、めっちゃおしゃれです！
いつもどこで買ったの？？って個性的なお洋服を着ています＾＾
 
 
ななみん！
 
ななみんの持ってる独特な雰囲気が好きです(o^^o)
洋服とか音楽とか色々センスがいいな～って思います（ミ＾o＾ミ）☆
実は陰の努力者で、自分の考えとかすごい持ってて、前にななみんとお仕事でホテルに泊まったとき、すごい語ったね（笑）
色々かっこいいなーって尊敬するとこが多いです。
 
 
 
みんな、おめでとう＼( ^o^ )／だいすき！
 
 
 
 
 
 
 
 
 
 
今日の乃木坂ってどこ？では、
「 13日の金曜日 」のPVが
初公開されます！
 
もうこの曲めちゃくちゃかわいいです！ダンスもかわいいし、PVもめちゃくちゃかわいい！
乃木坂史上最高に好きな曲です！
ぜひ、みてください( ^ω^ )☆ミ
 
 
 
さゆにゃん
