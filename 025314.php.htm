2015/1019Mon井上小百合ヽ(｡･ω･｡)のら

 
 
 
 
昨日は、個別握手会
有難うございました
 
 
色んなお話ができて幸せでした○゜
 
 
もうすぐハロウィンってことで、
みんな色んな仮装してましたね( ﹡ˆoˆ﹡ )
 
 
 
私は、、前にモバメでコスプレ何がいいかなー？
って聞いたら、755やブログのコメントで
皆さんからお返事をもらって、
猫っていうリクエストが多かったので...
 
 
はい。(///)照
：Flower
：evelyn
 
 
 
尻尾もついてます
 
 
ホントは首輪の鈴もあったんですが、
恥ずかしくて付けませんでした（笑）
 
のら猫ってことで
飼い主さん募集中です。(   ˙-˙   )
 
 
 
 
髪型は、
 
1部2部：ツイン。
 
 
 
 
 
3部4部：ハーフツイン。
 
 
まいまい黒猫と、さゆ白猫( =´ω`= )
 
 
 
 
つんつん
 
 
 
 
お知らせ〜。
 
 
今週の、｢週刊プレイボーイ｣さんの
『乃木坂46物語』、ぜひ沢山の方に
読んで頂きたいです。
 
色々とお話させて頂いて、改めて
考えることも沢山ありました\(﹡ˆᴗˆ﹡)/
 
 
 
今週23日(金)｢MUSIC STATION｣さんに
出演させて頂きます。
前回は、大阪で舞台に出演していて
出られなかったのでとっても嬉しいです。
ぜひ見てください(﹡ˆˆ﹡)✡
 
 
 
24日(土) 「乃木坂46 SHOW! ｣
(BSプレミアム 午後23時15分から)
 
メイドさんコントシリーズ、
また呼んで頂けて嬉しいです(*^_^*)
 
今回は、電気屋さんになりました！
やっぱりこの衣装何度着ても恥ずかしいです、
 
良かったら見てください( > <  )
 
 
 
では！！！
 
 
 
                     さゆヽ(｡･ω･｡)正義
