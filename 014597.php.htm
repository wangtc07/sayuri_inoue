2013/1013Sun井上小百合ヽ(｡･ω･｡) さい ごー

 
 
 
今日は、幕張にて握手会でした( ﹡ˆoˆ﹡ )
 
6th最後の個別握手会！
 
そして、自分にとっては、
18歳 最後の個別握手会！
 
 
来てくださったみなさん、有難うございました！！
 
では、早速レポートgo ⊂( ^-^)⊃
 
 
１部は、白と水色のもふもふワンピース♩♩
 
髪型は、おろしマキマキ＊
このリボン、こないだ川後pと
ディズニーに行ったときに買った
お気に入りのやつなのっ(#^.^#)きゅん
 
 
２部は、猫耳ヘアーっぽいやつに、
うさちゃんピンをつけたよっ(o^^o)◎
 
 
でも、なんだか羊さんみたいになっちゃった（笑）
もうなんの動物なんだか（笑）
 
もふ(＾ω＾)(＾ω＾)(＾ω＾)もふ
 
 
３部は、私がデザインしたBLT 
Tシャツに、黒のショーパン☆ミ
 
Tシャツは、安定のイン！！！笑
理由は、ダンスのとき踊りやすいのと、落ち着くから。┗(^o^ )┓<33
 
髪型は、たまこまのあんこちゃんヘアーにリボンっ（ミ＾o＾ミ）
 
 
４部は、本当は違うお洋服持ってきてたんだけど、みなさんからのリクエストで、またTシャツを着ました！
髪型も、リクエストのツインテール。
 
 
 
どの部に来た人もみんな楽しんでほしいから、全部変えてみました( ﹡ˆˆ﹡ )
 
そんな感じでした！！！
 
 
  ー
 
 
そして、
 
みなさんへ。
 
 
何度もごめんね(´,,･ω･,,`) ♡
 
今日は、6th個別を終えて、みなさんに
有難うとごめんなさいを伝えたいのっ。
 
 
さゆにゃんレーンに いっぱいになるまで、沢山の方が会いにきてくれて、嬉しかったよ( ﹡ˆoˆ﹡ )
すっごく、大好きで、大切です。
 
 
本当は、そんなみなさんを
心配させたくなくて、
自分が落ち込んでたらダメだって
精一杯、前向きにいよう と思ってたんですが、
 
残念だったね。って言われる度に、
ああ、自分のせいで みなさんを悲しませてしまったんだな...って。 
選抜発表後、初めて直接みなさんとお話して、改めて実感しました。
 
自分は、全然 大丈夫なんですが
みなさんの気持ちを考えると胸が痛くて...それがツライ。ごめんね、
 
気を使ってくださって、たわいも無い、楽しいお話をしてくださった優しい
みなさん。。
応援するから支えるから って、
本当に有難い言葉を言ってくださった
みなさん。。
 
そんなみなさんに対して、精一杯の
「 有難う 」と、「 頑張る 」  という言葉しか、言えなかった...  
 
一所懸命頑張ったからといって、必ずしも選抜や八福神に入れるわけでもないけれど、、
ただひたすら頑張ることしか出来ない
自分が申し訳なくて仕方ないです。
 
でも、そういう世界だということも重々承知しているし、今日、みなさんと握手して、
例えどんなに見えない努力であったとしても、諦めずにいようって思えました。
有難うございます(＿ )
 
自分の夢は、自分の中でしっかりとあり、いつかはこうなる! っていう目標もあるのですが、
その前に、私は、誰かに勇気や元気を与えたいと思って、アイドルになったので、みなさんの悲しむ顔は見たくないのです...
が、そういうものを生み出してしまって
本当にごめんなさい。
 
みなさんが、笑顔になれるんだったら
自分の辛さなんてへっちゃらだから、
サユリストさん、みーんなの笑顔がみたいなー♪
 
 
お話長くなっちゃってごめんなさい！
 
 
今日私いないけど（笑）
乃木どこも見てねっ（ミ＾o＾ミ）
 
 
さゆにゃんヽ(｡･ω･｡)正義！
