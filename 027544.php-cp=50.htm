2015/1209Wed井上小百合ヽ(｡･ω･｡)近況

 
 
 
昨日は、らりんとろってぃーと
ご飯食べに行ってきたよー( ﹡ˆoˆ﹡ )
お腹空いてたからモリモリ食べました（笑）
美味しかった〜
 
 
もふ〜
 
寒いのが苦手な私のために、
スタッフさんがギンギラのやつを
ぐるぐる巻にしてくれました（笑）
 
あったかかったです（笑）
 
 
 
イエイ♪
 
なんのポーズか忘れた（笑）
 
あ！おじいちゃんのポーズだ！( ･ิω･ิ)
 
 
 
最高ー！
 
 
 
 
 
反射しまくって、全っ然
ポスター映ってないや！ごめん（笑）
 
『リボンの騎士』大千秋楽おめでとう！
大阪まではいけなかったけど、
いくちゃん、れいか、
本当にお疲れさまでした
 
 
 
 
かずみんの隣確保
そしてくっついてくる真夏( ¨̮  )
 
みんなわちゃわちゃしてる（笑）
 
 
 
最近、目まぐるしく色んな事が
ありまして、毎日感謝です。
 
色々と報告しなきゃいけないこと
沢山あるんですが、ここいらで
なんてことない日常ブログを
上げてみました( ¨̮  ) 笑
 
 
 
  ー
 
さて、ここから、
お知らせたくさんです( ﹡ˆoˆ﹡ )
 
 
 
9日 (水)    ※本日！！！！
＊『別冊CD&DLでーた My Girl vol.7』
 
＊『ボム』
 
 
マイガールさんでは、
お誕生日を迎えるにあたって、
これまでの人生を振り返りました
 
(他にも、蘭世が出てます！）
 
 
ボムさんでは、生駒ちゃんと
グラビアをさせて頂きました(*^_^*)
 
個人的には、みかんを食べて
「すっぱぁぁ｣ってなってる
写真がお気に入りです（笑）
 
2人とも12月生まれなので、
お誕生日も祝って頂き、
本当に嬉しかったです( ﹡ˆoˆ﹡ )
有難うございました！
 
(かりんちゃん、純奈、ろってぃー、
きぃちゃん、蘭世、らりん、みおな、まあやも出てます！）
 
 
 
 
14日 (月)
＊ 19:00〜20:54 TBS系
『第48回日本有線大賞』
 
 
お誕生日は「両親に感謝する日｣と
言ってきましたが、偶然にも
この日、お誕生日を迎えます。(﹡ˆᴗˆ﹡)
 
乃木坂46が、このような番組に
呼んで頂けて、本当に光栄です。*
最早両親だけではなく、沢山の人や
色々なことに、感謝しながら
この日を迎えたいと思います。
 
 
 
16日 (水)
＊ 19:00～21:54 フジテレビ系
『2015FNS歌謡祭 THE LIVE』
 
 
 
19日 (土)
＊『PF（ポーカーフェイス） Vol.2』
 
 
まりかと表紙をさせて頂きました。
普段のイメージとは全然違う
メイクやファッションを
ぜひ見てください♪♪
 
(かりんちゃん、ろってぃー、ゆったん、まいちゅん、蘭世、まあやも出てます！)
 
 
 
24日 (木)
＊ 『street Jack 2月号』
 
 
"あの連載" に登場させて頂きます(﹡ˆˆ﹡)
 
撮影時、牛乳が好きという雑談をしてたら、
差し入れにわざわざ牛乳を
買いに行ってくださって、
さらに、美味しいとオススメの
"練乳鈴カステラ"も
買ってきてくださいました！
 
初めてたべたのですが、
この鈴カスと牛乳のコラボが
堪らなく美味しいことを発見し、感動（笑）
そのあとメンバーのみんなに、
大量に持ち帰って差し入れしたら
見事に完売でした！（笑）
 
 
あと、少し早いお誕生日
プレゼントも頂きました(  *ω*  )嬉
 
本当に色々と感謝です！
 
 
(ななみんも出てます！)
 
 
 
25日 (金)
＊  19:00〜23:10 テレビ朝日系
  『MUSIC STATION SUPER LIVE 2015』
 
＊ 『FLASHスペシャル』
 
 
フラッシュさんは、
万理華と、ひめたんと、3人で、
ほのぼのした私たちらしい
グラビアを撮っていただきました！
 
 
この日、携帯電話と間違えて、
テレビのリモコンを
持ってきてしまいました（笑）
 
色も大きさもだいぶ違うのに（笑）
相当寝ボケながら準備してたのかな...
 
それにしても、そんな漫画みたいな
ボケ、本当にあるんだ！って
みんなに笑われました。（笑）
 
 
(飛鳥、まいやん、なぁちゃん、ななみん、さゆりん、かりんちゃん、純奈、きいちゃん、伊織、琴子、まいちゅん、絢音、蘭世、未央奈、れなち、みり愛たんも出てます！)

(メンバー全員出てます！)

ぜひ！！！！！
