2014/1024Fri井上小百合ヽ(。・ω ・。)ス トレート〜

 
 
こんばそわヽ(^ω^)ノ
珍しくストレートにした日の写メ（笑）
 
 
アンダーライブのこと...
何度も書いてみたりするんだけど
文字にすると、どうも薄っぺらくなっちゃう感じがして、うーん違うなあ〜って、文章苦戦してます（笑）
濃すぎて どこから何から話せばいいかもまとまらなくて（笑）
 
次の記事あたりでちゃんと書くので少々お待ちを！(。> <。)あしからず...
 
 
 
あっ先日、若さまの舞台を観に行ってきました(* ^o^ *)！
 
円形劇場は初めてだったので、
このまん丸の舞台からどうやって話が進んでいくんだろう...
って、ドキドキワクワクでした＾＾
 
舞台が始まると、最初は不思議な感覚でしたが
だんだん世界観がわかってくると楽しくって、
途中からツボにハマって笑いが止まらなかったです（笑）
とにかくじわじわきて。。
もう、劇中歌が面白くて(>_<。)泣www
頭から離れなくなっちゃいました（笑）
 
 
若さまはとにかく可愛かったです／(^o^)＼笑
演技もさすが安定してました！
 
千秋楽まで怪我なく頑張ってね(  *^_^*  )
 
 
 
あ、あと、のぎ天。
他仕事があって、井上 3週間分くらい出てません(´・ω・｀)すまないです。
 
でもみんなとっても可愛いから
見てあげてください／(^o^)＼よろ
 
 
  ー
 
 
あ、モバメのお話し！
 
ハンバーグ作りました(* ^o^ *)○゜
 
 
チーズINです♪
 
お箸と比べてみるとわかるけど
小さめに作ったので、
結構食べれました( # ´ω` #   )満腹〜っ
 
 
 
『  レシピ  』
 
◎1   卵がなかったので、その代わりに
コーンフレークを潰してマヨネーズと豆乳を混ぜたものを
つなぎとして使いました。
 
 
◎2   玉ねぎはあらかじめ炒めて冷ましておく。
 
 
◎3    1と2とお肉を混ぜる。
自分の手で温めてしまわないように、冷やしながらコネるのがポイント！
チーズを包んで丸めて真ん中をへっこませる。
 
 
◎4   フライパンで少し焼き目を付けたら、
ワインでコトコト蒸し焼きにする。
(こうすることでお肉が柔らかく出来上がります)
 
 
〜ソース〜
 
◎  ずーっと欲しかった色んなお肉料理に使える万能なタレを手に入れることが出来たので、
そのまま4の汁の中に、そのタレとソースとケチャップを入れて、軽く煮詰めて完成☆
 
 
 
「レシピも教えてー」ってコメントも多かったので、今回から書いてみたよ(   ,,・ω・,,  ) 
 
結構、お家にあるものでパッと作っちゃったりするので、
さゆのオリジナルレシピですけど
良かったら参考にしてみてねっ♪
 
 
ー
 
質問にこたえるよ(ミ＾o＾ミ)
 
 
▼  Name :  るんるん  さん
 
◎  アンダーライブ中メンバーとご飯行ったりとかはしてたの？
 
＊  川後Pとめっちゃ行ったよ（笑）
 
あ、でも今度、みんなでお疲れさま会したいねって話してるよ\(^o^)/
ホントこの期間でより一層絆が深まった！！
 
 
 
▼  Name :  ホッシー☆  さん
 
◎  さゆはお母さんの手料理で何が一番好き？
 
＊  うーんいっぱいある(。> <。)
中でも、かぼちゃの煮物と茄子の味噌炒めが好き！
あーでも、肉じゃがも好きだし、天ぷらも好きだし...
 
お母さんの手作りこんにゃくも美味しい！
お正月に作る手作りのお餅も美味しい！
 
結局みんな好き（笑）
 
 
 
▼  Name :  ひろぽん  さん
 
◎  さゆは、ハンバーグにかけるなら何が好き？
 
＊  デミグラスが好き！
でも時々、さっぱりしたお醤油ベースのも作るよっ
 
 
 
▼  Name : バチバチくん  さん
 
◎  なんとなく思ったんだけど、握手会で何度か着替えるけど、荷物多くならない？
 
＊  多くなるよっ笑！休憩時間もなくなる笑
 
でも、何度も来て下さる方とか、毎部来て下さる方もいるから、
みんなに楽しんでもらいたいなって思って(* ^o^ *)！
 
 
 
では！！！
 
 
さゆヽ(。・ω・。)正義！
