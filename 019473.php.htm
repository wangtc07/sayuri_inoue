2014/0720Sun井上小百合ヽ(。・ω ・。)ほ っこり。

 
 
最近、風鈴がほしいさゆです。
 
あー夏だなぁーって、風を感じたいです。
 
どっかいいとこないかね？
風鈴買いに行こうかのぉ(。-｀ω´-)ほふ
 
 
あ！日光に行ったら、蝉のなきごえ聞こえました！
ヒグラシがないてました！
あと、うぐいすもないてました！
よきかなー
 
 
 ー
 
 
最近見た、ほっこり特集〜\( ^^ )/♪
 
(  なんとなく写真におさめたものの、
載せる機会が無さそうなので、
ここで載せときまーす笑  )
 
 
お花にちょうちょがとまってました。かわいい(  * ^_^ *  )
 
 
 
あじさい。よく見るとこっちにはミツバチがとまってる。
 
 
 
おっきな百合の花。
井上家のお庭のユリは小さい。
だから小百合なんだよねっ
 
 
 
よきかなーよきかなー
 
 
 
お花屋さんのいとこが、
お母さんのお誕生日にくれたお花 ＊
キレイ(* ^o^ *)
 
 
みんなもほっこり出来たかな？？*
 
 
 ー
 
質問にこたえるよ(ミ＾o＾ミ)
 
 
▼  Name :  たーく さん
 
◎  いつも私服可愛いけど買い物する時のポイントは何？？
 
＊  
モデルさんが着てる写真とか、口コミとか、色々調べていいなと思ったやつを買ってます。
 
体型がコンプレックスなので、出来るだけ体のラインが見えないのと、露出控えめのを選ぶのがポイントです。
 
 
 
▼  Name :  おじゅん さん
 
・さゆりにとって千葉のイメージは？
埼玉のライバル？（笑）
 
＊ 千葉は... かずみんのイメージ（笑）
 
よく、かずみんと「千葉の端っこ出身」と、「埼玉の端っこ出身」の話で盛り上がるから（笑）
 
あ、明日は幕張メッセだ*\(^o^)/*
千葉県の皆さんよろしくー
 
 
・足のサイズは何センチ？
 
＊  23.5だよ(´。・v・。｀)
普通かな？
 
 
 
▼  Name :  あまちゃん さん
 
◎  にゃんさゆに質問！
最近暑くて寝苦しいよー...
暑くて寝れないときは何か工夫してることあるかな？
 
＊  うーん、、本を読む（笑）
 
最近、お母さんから本貰ったんです！
「人生はワンチャンス！」と「人生はニャンとかなる！」っていう、
ひたすら犬と猫がかわいい本（笑）
 
 
 
▼  Name : 部長(カピバラ太郎) さん
 
◎  質問していい？(*^^*)
最近さゆが可愛過ぎてご飯が喉を通りません。
どんぶりに5杯しかたべられません。
どうしてくれるんですか？(笑)
 
＊  な、なぬ(°_°) 申し訳ない...
 
 
ん、めっちゃご飯食べてない？笑
 
 
 
▼  Name : さゆりすとあらし さん
 
◎  さゆどういうイヤホンつかってるの？
 
＊  あ、最近、新しいの買ったの！
ヘッドフォンなんだけどね、
真っ白の！ビーツの！(o^^o)えへへ
 
 
 
▼  Name :  しょうた(ツッコミ担当) さん
 
◎  北海道行ったら行きたいのは、海？山？どっち？
 
＊  山ーっ *\(^o^)/*
 
 
 
▼  Name :  にすはら  さん
 
◎  質問:して欲しい質問はなんですか？笑
 
＊  えー？  んー(´× ×`)なんだろ？
 
聞きたいこと、何でも聞いてくれていいよ( *＾＾* )
できる限り色々こたえまーすっ
 
 
 ー
 
 
さて、明日は幕張で握手会です*\(^o^)/*
 
せっかく夏だから...
 
あれを着ちゃおう！と思います♪
 
 
はて、自分で着れるかな？心配だなあ（笑）
 
 
来てくださる皆さん、よろしくですっ
待ってます(* ^_^ *)
 
では！！！
 
 
 
さゆヽ(。・ω・。)正義！
