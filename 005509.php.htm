2012/0623Sat井上小百合千変万化。 ＋＋

 
 
昨日は、まりかとみなみが
私のお家にお泊りしにきました(*^o^*)わーい
 
 
おとまり～  おとまり～っ
 
 
 
おふざけ 沢山したよっ ＼^o^／  ！（笑）
 
 
 
わちゃわちゃっ )^o^(  )^o^( 爆
 
 
この写真を撮るのにね、何回も笑っちゃって撮れなくて
大変だったんだよー（笑）（笑）
 
でも、ほんと楽しかった！(o^^o)
 
 
２人とも、また来てね～(=^ω^=)
 
 
 
今日は、生駒っちとおしりを
可憐にさわる戦いをして遊びました(*^ω^*)
 
生駒っちと おしり合いになれたぜ！へっへへーい！
 
 
質問回答いきまーす！
 
 
○にゃんの今一番やりたい事、やってみたい事は？
 
今やりたいことは、
もっと勉強がしたいです ！＊
 
 
やってみたいことは、資格を取ること...かな(*^_^*)
 
 
○和菓子と洋菓子でそれぞれ１番好きなのを教えて！
 
和菓子なら、あんみつが好きです！
 
洋菓子なら、バタークッキーが一番好きです(*^_^*)
大好きな牛乳やカフェオレと一緒に食べるのが(#^.^#)
自分でも作ったりするけど、お母さんに頼んで実家から送ってもらうくらい好きです！
 
 
○さゆにゃんはサーティワンに行ったら頼むのやっぱトリプル？(笑)
 
食べるの遅くって溶けちゃうから...(*´pω･｡)うむむ
シングルで十分かもですっ (> <  )
 
 
○さゆにゃんの好きなたまご料理は何ですか？？(^w^)
 
ホットケーキと、ドーナツ！！
クッキーにクレープに...（笑）
 
でも、お母さんの作るたまごスープが一番です。(*^_^*)
 
 
○さゆにゃんって嫌いな食べ物とかある!?
 
お魚が苦手です(>_<)
 
とくにお刺身とかの生魚が苦手で（ ;  ; ）
 
あとは、とろろとかも
食べられないです（ ;  ; ）かゆかゆ
 
 
 
 
ということで、今日も一日 
お疲れさまです
 
 
明日もいい日になりますようにっ＊＊
 
 
 
 
         さゆにゃんヽ(｡･ω･｡)
