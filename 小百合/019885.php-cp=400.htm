2014/0816Sat井上小百合ヽ(。・ω ・。)た こ焼きー

 
 
こんばそわヽ(^ω^)ノ
 
先日は、「FNSうたの夏まつり2014」に出演させて頂きました！
 
音楽の楽しさ素晴らしさを改めて感じた一日でした。
とっても楽しかったです。
 
もっともっと沢山の舞台で伝えていけたら素敵だな。
 
さて、明日からはいよいよ
全国ツアーの幕開けです！！
 
沢山の方たちが私達のために素敵な
 
最近、毎日筋トレしてます。仕事、リハの合間、家とかとか。
体力つけなきゃー(* ^_^ *)あと痩せたい笑
大好きなお風呂も、最近はゲルマニウムの入浴剤を入れてます。
お風呂入る前にいっぱいお水を飲んで、ちょくちょく休憩しながら1時間以上は半身浴をします。
めっちゃ汗かいてスッキリするんだーｐ(*^-^*)ｑ
 
 
あ、でも今日たこ焼き食べちゃった（笑）
 
明日たくさん動くから許して笑
 
 
大阪、盛り上がっていこーーーう
*\(^o^)/*
 
 
 ー
 
質問にこたえるよ(ミ＾o＾ミ)
 
 
▼  Name :  てっかん  さん
 
◎  でかい花火と線香花火はどっちが好き？
 
＊  線香花火が好き(* ^o^ *)！
実は、こないだメンバーのみんなと線香花火やったの！
誰が一番最後まで長く消さずにいられるかって勝負して、楽しかったなあ。
 
 
▼  Name :   スーシー   さん
 
◎さゆは同期のなぁちゃんとキャプテンとは仲良しなんですか？また、一緒に食事に行ったりしないのですか？
 
＊  仲良しだよー♪
昨日も、なぁちゃんと、らりん、ゆったんとご飯に行ってきました(* ^o^ *)！
 
れいかとは、ユニットとかで歌割りがペアのことが多くて
個人的にれいかの歌声とダンスが大好きだから嬉しい(* ^_^ *)♪
 
 
▼  Name :  ホルン  さん
 
◎  夏コンで、思い出に残っている曲とかありますか？
 
＊  いっぱいあるけど......リバーダンスかなぁｐ(*^-^*)ｑ
 
 
▼  Name :   まさまさゅにゃん  さん
 
◎  さゆにゃんの夏休みの目標教えてー(*≧∀≦*)
 
＊  夏休みは無いけど今年の夏も
日焼けしないことと、
ツアーがんばる☆
 
 
▼  Name :  じゅんじゅん☆彡  さん
 
◎ゲスの極み乙女。っていうバンド知ってる？
 
＊  好きです*\(^o^)/*
最近、みさ先輩もハマってくれたみたいで、そのこと3回くらい話してくれた（笑）
 
 
さゆヽ(。・ω・。)正義！
