2014/0801Fri井上小百合ヽ(。・ω ・。)お めでとです

 
 
こんばそわヽ(^ω^)ノ
 
最近、ボケがなおらないさゆです。
 
今日なんて、お仕事終わりで
せいらりんとゆったんとご飯に行ったんだけど、
ごちそうさまして、荷物持たずに
レジのほう行こうとしちゃって
2人が「忘れてるよ！」って気付いてくれたから良かったけど、
危ない危ない(´^ω^`;)
 
気をつけなきゃなー(-´ω｀-。)
 
 
 ー
 
 
今日で7月も最後ですね！
 
タイミングを逃して、7月生まれのみんなのお誕生日、ブログで祝えてなかったから
遅くなっちゃったけど、ここでお祝い！(。>_<。)すまぬっ
 
7月17日 ひなこちゃん ＊
7月20日  ゆったん ＊
7月23日 ろってぃー＊
7月27日 れなさん ＊
 
おめでとうーっ*\(^o^)/*ぱおぱお
 
 
 
れなさんには、今日久々に会えたので、
ケーキサプライズをすることが出来ました(* ^o^ *)
 
乃木坂では恒例なんだけど、
れなさんは初めてだからビックリしてくれてて、嬉しかったです＊＊
 
 
 ー
 
質問にこたえるよ(ミ＾o＾ミ)
 
 
 
▼  Name :  うさみちゃんず さん
 
◎  黒石さんとかブラックひめたんとか、そのギャップキャラ、ファンに好評ですが...さゆニャンもたまーにはちょいブラックな「黒百合さん？」になったりする？
 
＊  んー......電車とかで優先席に座ってる若い人とか、歩きながら携帯してる人とか見ると
「んっo(｀_´)o」ってなったりする。
 
あと、ダンスとか歌とか完璧にできないと自分にイラついて 「んっo(｀_´)o」ってなる（笑）
 
 
 
▼  Name :  にくきゅうくんくん丸 さん
 
◎  ネット通販の達人のさゆさゆは、買った服が思ってたのと違った場合どうしてますか？
 
＊  まあコレでもいいかー(* ^o^ *)
ってなる（笑）
 
当たりハズレも、届くまでの楽しみなんです♪
 
サイズとかが無理だなってやつは、妹とかメンバーが欲しいって言ったらあげちゃいます。
 
 
 
▼  Name :  乃木隊員 さん
 
◎  トッキュウジャーの『乗り換え』が乃木坂メンバー内で出来るとしたら誰に乗り換えます？
 
＊  ななみん*\(^o^)/*
 
ショートの女の子が好きだから。
 
 
 
▼  Name :  take14 さん
 
◎  口の中の水分がなくなった時、
バームクーヘンを食べますか？
それともコッペパンを飲みますか？
 
 
＊  飲む！？コッペパン、飲む！？笑
 
えーっと...飲むのは辛いから、、
バームクーヘン食べます(。> <。)笑
 
 
 
▼  Name :  さゆゆは天使 さん
 
◎  さゆは一日に何回歯磨きしますか？
 
＊  4回ですっ
 
朝起きたあと、朝ごはんのあと、
お昼ごはんのあと、夜ごはんのあと
です*\(^o^)/*
 
 
 
▼  Name :  みつき さん
 
◎  バニラアイス以外得意じゃないらしいけどバニラアイスにクッキーとかが混ざってるようなアイスは食べられるのかな？
 
＊うーん、食べれはするけど
自分からは食べないな〜
出来れば、別々で食べたい...(´^ω^`;) 笑
 
それぞれ、そのままが好き。(*^-^*)
 
  ー
 
 
では！！！
 
 
さゆヽ(。・ω・。)正義！
