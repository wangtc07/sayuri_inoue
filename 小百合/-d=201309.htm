2013/0930Mon井上小百合ヽ(｡･ω･｡) 僕と あくしゅー

 
 
 
おすわりにゃん。
 
 
 
昨日は、名古屋にて握手会でしたあ( ﹡ˆoˆ﹡ )
 
結構混んでるんじゃないの？ってみなさんから言われてビックリしたけど、
レーンの外まで列が折り返すくらい、沢山の方がきてくださって... 
本当に嬉しかったです(o^^o)◎ てへへ
 
 
きてくださるみなさんに 感謝して、
誠心誠意、握手したいなって
いつも思ってます( ﹡ˆ ˆ﹡ )♡
 
でも、並ぶ時間長くて疲れさせちゃってないかなとか心配で...
それに、あれだけ混むと、一人一人の握手も短くなっちゃうし...
なんか本当に申し訳なくて、
やっぱり３部じゃ足りないなあって
思いました...( >_< ) 
 
 
なにより、みなさんのことが大好きだし、
お話できることが楽しいので、
もっとお話したい！ って、
こんな照れ屋で口下手な私が言うの
ビックリすることだけど、(´^^`; )
 
 
本当に...
 
本当に、みなさん感謝ですペコリーノですトスカーナです（笑）
 
 
 
あっ
わたし、謝らなきゃいけないことがあってね...
 
昨日、みり愛ちゃんが、休憩時間 一緒に写真撮ってもいいですか～？って、話しかけてくれて、
大丈夫だよ♪有難う～って言ったのに、
時間がなくなっちゃって、
写真、撮れなかったんです(´+ω+｀)
それから会えずじまいで...
ほんとにごめんね(*´pq｀)
 
まだ、連絡先を交換できていないので
この場を借りてお詫びを... 
 
また一緒に写真撮れたら嬉しいなっ＊
 
 
  ー
 
そして、みなさん、髪型のこと
色々お返事くれて有難う ( ﹡ˆoˆ﹡ )
 
握手前に、コメント読んだよ～ ♪
色んなリクエストがあったんだけど...
その中でも若干 多かった、
 
前髪は、くるんっ  で、
髪は、おろし！ に決めた( ^o^ )☆ミ
 
毛先をちょこっと内巻きにしてもらったよ (* ^_^ *)
 
 
いつもと違う感じ ♩♩
こんなのもどおかな～( ﹡ˆoˆ﹡ )照り
 
 
 
 
 
ちょっと大人っぽい...？ ( 〃 ˆ ˆ 〃 )
 
 
 
４部からは、一年ぶりくらいの、
ハイツインをっっヽ(^ω^)ﾉえへ
 
嬉しいことに、最近 私のことを真似してきてくれる女の子が増えてきて...ホント嬉しくって
いやぁ女の子みんな可愛すぎて
女の子だいすきヽ(^ω^)ﾉヽ(^ω^)ﾉうへ
正義ヽ(^ω^)ﾉヽ(^ω^)ﾉヽ(^ω^)ﾉうへへ
 
それで、この 超高めの位置でする
二つ結びのさゆにゃんヘアーを
久しぶりにやってみちゃいました┗(^o^ )┓ペコリーノ
 
 
 
珍しくブログが長くなってしまったのでヽ(^ω^)ﾉ笑
読むのも大変だと思うから、私服は
次のに載せたいと思いますっヽ(^ω^)ﾉ＊＊
 
それでは～～っっっ┗(^o^ )┓<33┗(^o^ )┓<33┗(^o^ )┓<33┗(^o^ )┓<33
 
 
 
◎  サックス吹ける女性ってホントかっこいいと思います！
自分はジャズもロックも好きでバンドでベースボーカルをやっているのですがさゆにゃんは好きなロックバンドはありますか？
 
＊  いえいえ 恐縮ですっ ；；
お！ わたしも高校で
軽音楽部 でした (* ^o^ *)！
ただ、ボーカルの私の声に合わせた曲をコピーしてたので、ロックじゃなかったんですけどね(´^ω^`;)笑
 
好きなバンドは、
Galileo Galileiさん＊
SEKAI NO OWARIさん＊
サカナクションさん＊
相対性理論さん＊
 
ですっ(o^^o)◎
 
 
 
さゆにゃんヽ(｡･ω･｡)正義！
