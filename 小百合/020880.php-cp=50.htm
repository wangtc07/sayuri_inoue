2014/1019Sun井上小百合ヽ(。・ω ・。)さ ゆデビル

 
 
昨日は、パシフィコ横浜にて
個別握手会でしたー(  *^o^*  )
 
今月はハロウィンということで、
みんな色んなコスプレをしていました！
 
ということで、私も便乗して〜
 
 
 
1部 ＊＊
 
さゆデビルー
 
モバメやブログで質問したら、
「いつも天使って言われてるから、逆に悪魔とか見てみたーい」
って意見が多かったので、皆さんの意見を採用して
黒い羽の生えたブラックさゆになったよー(* ^p^ *)！
 
陽菜ちゃんと一緒に買いに行ったん♪
 
 
 
2部、3部 ＊＊
 
髪型をハーフツインにしました♭♭
 
 
因みに、楽屋はかずみんと隣の席でしたー(o^^o)
 
かずみんが、さゆデビルの楽屋の様子をブログに載せてくれたよ（笑）
 
 
 
4、5部は、みさ先輩とまいやんと買ってきたコスプレで、
コナンくんをしました(  *^o^*  )笑
 
恥ずかしかったのでここには載せません///笑笑
 
来てくれた方だけの特権〜（笑）
 
 
  ー
 
 
休憩時間に、生誕祭もありました！
 
あみ、お誕生日おめでとー(*^o^*)
20歳だねー！！
 
3年前、レッスンに遅れてきたあみが
「今これ何やってるの？」って聞いてきたのが、初めての会話でした（笑）（笑）
そのときは、すっごくクールなイメージだったけど...  いつからか
ふざけ合える仲になってたね（笑）
 
いつも至近距離で会話するから
チューしてるのかと間違われるけど（笑）
これからも、あみが大好き！！！！
 
 
 
 
そして、握手会が終わったあとは、
卒業セレモニーがありました。
 
 
りさこちゃん、きょうちゃん ＊＊
 
 
これから、色々な道に進んでも
二人の未来を応援します ○゜
 
一年半、有難うね、
お疲れさまっ。
 
本当に、卒業おめでとう！( >_< 。 )
 
 
 
ねねちゃんは、今日のアンダーライブでほんとのほんとの卒業でした......
 
アンダーライブについては、濃すぎてまだまとまらない( >_< 。 )
 
あとでゆっくり書くねっ♪
 
みんな大好き！！！！
 
 
 
 
今日放送の、らりんとかりんちゃんと出てる『音ボケPOPS』
良かったら見てくださいっ(  *^_^*  )＊
 
 
では！！！
 
 
さゆヽ(。・ω・。)正義！
