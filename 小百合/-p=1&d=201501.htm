2015/0131Sat井上小百合ヽ(｡･ω･｡)温泉行きたい 、

 
 
本日は、東京ビックサイトにて
個別握手会でした！
 
来てくださった方、有難う*\(^o^)/*
 
 
では、来れなかった方にもレポート！
 
 
1部＊
ストレートおろし。
 
 
 
2部＊
ツインテール。
 
 
 
実は最近、お仕事で前髪をパッツンにしたんです。
でも幼くなっちゃってすんごく恥ずかしい...(｡•́ - •̀｡)しゅん
 
 
 
服装はこんな感じ！
 
 
 
3部＊
ポニーテール。
 
 
 
4部、5部＊
ネコちゃんヘア。
 
4部のあとの休憩時間に眠っちゃって、
5部変えれなかった（笑）
 
 
全部、
 : MAJESTICLEGON
 : INGNI
 : NIKE
です。
 
 
初めましての方も沢山居て、嬉しかったです(っ´ω`c)
 
ライブでのキリッとした私を見て握手に来てくださった方は、
普段のふわっふわなギャップに
びっくりするでしょうね。（笑）
 
逆に、この普段のふわふわ状態を知ってる方には、歌って踊ってる時のキレッキレな井上に
びっくりするようです 笑
 
 
まったく、モグモグとかモグにゃんとか
散々いじりやがって(º﹃º )
 
こんな私だけど、少しでも皆さんのこと
癒せたかなー？？
 
 
 
  ー
 
 
今日はひなちまのお誕生日です！
 
おめでとう！！！*\(^o^)/*
 
17歳、若い！
素敵な一年になりますように！
 
 
 ー
 
 
お知らせ！
 
 
＊  BUBKAさんに、らりんとゆったんと載ってます(*^_^*)
 
チェックしてね！
 
 
 
＊  2月7日(土) 23:30〜 NHK BSプレミアム「乃木坂46 SHOW!」
 
あのカップリングをTV初披露と、
映像コントに人生初挑戦しました！
 
メイドさん着ました。恥ずかしい...
 
 
ぜひ、見てくださーい( *^o^* )
 
 
 
さゆヽ(｡･ω･｡)正義！

2015/0126Mon井上小百合ヽ(｡･ω･｡)もぐもぐ

 
 
乃木坂ってどこ？見てくださった方、
有難うございます！( *^_^* )
 
モグモグ恥ずかしい...(๑′ฅฅ‵๑)
 
 
よく、猫とか 冬眠前のリスみたいって言われるのはそういうことか...╰(　・´ｰ・｀)╯
 
でも、口が小さいのがコンプレックスで、自分としては結構一生懸命食べてたりするんですよ（笑）
 
 
いつもおっとりしてて、気付いたら寝てたり、
暇があれば、公園とか庭園に行って日向ぼっこしてるのが好きなんですけど、
↑多分、前世ネコ。（笑）
 
ときどき、突拍子もなくシュシュっと素早く行動することもあるんです！笑
 
ダンスや演技のときは豹変するようで、そのギャップにびっくりされることがあります。
 
 
どうやら、普段はフワーッとボーッとしてますが、
ここぞというときに入る スイッチが
あるようです(´・∀・｀)笑
 
 
 
まあそんなことより、成人企画をやって頂けてほんとに嬉しかったです*\(^o^)/*
 
実は、今までお魚が苦手で近寄らなかったんですが、
あの時 食べたお寿司があんまりに美味しくて感動して、
あの日からすっかりお寿司大好きになっちゃいました（笑）
 
 
それで、次の日も、その次の日も、お寿司食べちゃった（笑）
 
お魚食べれるようになったぞ！って
最近すごく張り切ってます。笑
 
乃木どこさん有難うございました！！
 
 
  ー
 
 
ニッポン放送 LIVE EXPO  TOKYO 2015  ALL LIVE NIPPON Vol.3に、
10thアンダーメンバーと研究生で
出演させて頂きました。
 
来てくださった皆さん、有難う！
タオルやうちわやサイリューム声援、とっても嬉しかったです！
 
 
こんな広い会場にゲストとして呼んで頂けるなんて、有難かったです。
 
オープニングを務め、約一時間もパフォーマンスやトークをさせて頂き、
とても緊張しましたが楽しかっです！
 
自分としては、ラジオが大好きで、いちリスナーでもあるので、
本当に嬉しかったし、光栄でした。
 
「自由の彼方」も初披露！
自然と笑顔で歌ってました。
 
それから、こないだ沈黙の金曜日に出させて頂いてから、
アルピーさんがさゆにゃんって沢山呼んで下さって、嬉しかったです。
また一緒にお仕事出来たらいいなあ♪
らりんをよろしくお願いしますo(^_^)o
 
 
 
今回は代表して挨拶やMCなどを務めさせて頂きました。
率先して色んなことに取り組むようになって、この期間で、沢山のことを学びました。
自分の中で色んなことが変わったし、もっと変わりたいと思った。
 
11thは、また違った色んなことを沢山学ぶと思います。
自分がどんな変化を見せるのか、自分でもわからない。けど、楽しみにしててほしいし、楽しみたいな♪
 
なによりこのお仕事が好きだから(*^_^*)
 
 
今、今後の自分にとって一番大切な時間を過ごしている、
そう感じます。
 
 
 
もうこのメンバーで最後っての、寂しかったけど
でもやっぱり楽しかった！
 
 
 
 
 
わーいわーい！
 
 
  ー
 
 
 
お知らせ！！！！
 
 
24日発売の、SamuraiELOさん見てくださいっ♪
 
白いカーディガンにフワフワツインテールっていう
最近にしては、珍しい格好をしました（笑）
ので、見てくださいね o(^_^)o  照
 
 
 
  ー
 
 
Mステ見てくださった方、有難うございます！
今回は、ファンの方とライブ形式でメドレーを披露させて頂いて
とっても楽しかっです！(*^_^*)
 
また出たいなー！
 
 
 
  ー
 
 
いくちゃん、まいちゅん
ハッピーバースデー*\(^o^)/*
 
二人にとって、素敵な一年に
なりますよーにっ○゜
 
 
では！！

2015/0119Mon井上小百合ヽ(｡･ω･｡)背負う

 
 
昨日は、横浜にて個別握手会でしたーo(^_^)o
 
来て下さった皆さん、有難う！
 
 
 
そして
乃木坂って、どこ？の選抜発表を
見てくださった方、有難うございます。
 
自分はインフルエンザにかかってしまっていて、
参加することが出来ず、すみません( ; ; )
 
 
 
結果を知ってから、しばらく一人で色々考えましたが、
 
ポジションがどうであれ、
自分らしく、感謝の心を忘れず
これからも頑張るのみ！という考えに至りました(*^_^*)笑
 
 
 
今は、この場所で自分に任された
課題が沢山あるような気がしていて
 
 
それを、ちゃんと自分の手で切り拓いて
グループごと押し上げていきたいって思うんです。
 
 
 
10thは難しい立場ではありましたが、決してアンダーだからと言ってマイナスなことだけではなく、
 
むしろ、葛藤の中で得たものは
とても大きかったです。
 
すごく有難いことです...(__)
 
 
 
アンダーだからこそ見れた景色も沢山ありました。
 
笑顔も涙も、絶対一生忘れない。
 
そして、あの景色を糧にして
頑張りたい。
 
 
 
ブルーシアターの公演で
10/16、Wアンコールがかかった日
 
結成時から乃木坂を支えてきてくれたマネージャーさんが、メールを下さいました。
 
 
「この難しい状況で、素晴らしいセンターの役割を果たしてくれました。
ひとつ大きな物語を残したと思います。」
 
 
正直、毎日落ち込んで、自信を失ってばかりでしたが
この言葉に、少し救われました。
 
 
10th期間を通して、何より嬉しかったことは、
身近なスタッフさんやメンバーから
このような言葉を掛けて頂けるようになったことです。
 
ファンの方も、他推しの方や初めましての方が、
握手会などに来て下さったり、ブログにコメントして下さることが多くなって、
本当に有難かったです。
 
 
この期間、感謝してもしきれない程
とても沢山の方に支えて頂いたので
 
今度は自分が誰かのために
頑張らなきゃです。
 
 
いおりちゃん、初めての事ばかりで戸惑うこともあるだろうけど
私も経験したことのあるポジションだから、いつでも頼ってきていいからね！
 
 
まりか、飛鳥、本当におめでとう！本当に嬉しいです。
 
二人の分も、今度は私が背負うよ。
 
 
 
 
11thも、乃木坂46を
どうか宜しくお願い致します。
 
 
 
ただ一つ言っておきたいことは、
どんなことがあろうとも、
乃木坂に入ってから私の目標は変わっていません。
 
 
いつか、 グループの真ん中に立って、
もうひとつ、大きな物語を残したい。
 
 
 
さゆヽ(｡･ω･｡)正義！

2015/0111Sun井上小百合ヽ(｡･ω･｡)考えて行動

 
 
 
最初で最後の成人式。
 
人生でたった一度だけの大切な日を
 
昨日、乃木神社で、大好きな94年組のみんなと迎えることが出来ました。
 
 
ファンの皆さん、スタッフの皆さん、お父さん、お母さん、支えて下さる全ての方々
ここまで本当に有難う。
 
20年間生きてきて、こんな素敵な成人式ができて、幸せっ(﹡ˆ﹀ˆ﹡)♡
 
 
そして、同い年の新成人の皆さん、
御成人おめでとうございます！
 
 
 
振り袖は、白にしましたo(^_^)o
「小百合」という名前の由来になった、「小さい百合の花の色」！
 
だから、自分の中でのイメージカラーでもあるし、
応援してくれるサユリストの皆さんも
いつも白のサイリュームなどを持ってくれているので、
 
ずーっと、振り袖は白を着たい！って決めてました( *^_^* )
 
 
素敵な振り袖を着させて頂いて、
有難うごさいました。
 
 
 
みんなもすんごく可愛かったです！
94年組は、私の中で特別な存在(﹡ˆˆ﹡)
 
 
式典には、いつものイベントとは違って、皆さんがスーツを着てきてくださって、
なんだか親？みたいで嬉しかったです。（笑）
とくにサユリストの皆さんね！笑
 
そのあと、サユリストの皆さんと
記念撮影もしたけど、
ほんとに家族写真みたいだったね！......七五三の。（笑）
 
 
 
これからも、子供みたいに頼ってしまうこともあるかもしれないですが、
ハタチの井上小百合も、どうか宜しくお願いします。 
 
ほんとにほんとに有難うございました( *^_^* )
 
 
 
 
 
あみが色気づいてる。笑
 
 
ナタリーさんなどにも、成人式の様子が載っているので
見てみてねー(﹡ˆoˆ﹡)
http://natalie.mu/music/gallery/show/news_id/135919/image_id/352331
 
 
 ー
 
 
開店前の渋谷TSUTAYAさんに、メンバーのみんなと
おじゃまさせて頂きました！
 
すっぴんメガネー 
 
しかも、まりかが撮ってくれた写メブレてる（笑）
 
 
もう、天井から床までびっちり乃木坂で！
すごい！もうホント嬉しい！！
 
有難うございます( > <。)
 
 
明日までなので、皆さんも、ぜひ行ってみてください(*^o^*)
サインも書いてきたのでみてねー♪
 
 
では！
