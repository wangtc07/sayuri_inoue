2015/0211Wed井上小百合ヽ(｡･ω･｡)最近のこと話 しまーす

 
 
 
＊ 2/7
 
(   1stアルバム 『透明な色』
発売記念スペシャル個別握手会   )
 
好きな歌衣装を選んで着れるので、
755で皆さんからの意見が多かったやつを着ました！
 
 
1部 : 13日の金曜日
2部 : 制服のマネキン
3部 : ガールズルール
4部 : あの日 僕は咄嗟に嘘をついた
 
みりあとお揃った！
 
うへへ
 
うにゃにゃー
 
 
 
ガールズルールも懐かしい！
 
 
また みりあと被った！（笑）
 
 
歌衣装での握手会も新鮮で楽しかったですね(　ˆᴘˆ　)♪
 
いつもこれでステージに立ってるから、
パフォーマンスしてるときの私と
話してるときのフワフワさのギャップに
「不思議な感じー」って
いっぱい言われた(  ´^_^` )えへへ
 
 
 
＊  2/8
 
(  10thシングル「何度目の青空か？」
発売記念 個別握手会    )
 
私服紹介。
 : カスタネ
 : GU
 
 
シャツ襟♭
 
アイドルのときの自分とは違って、
普段はこういう服装のほうが多いです。
ゆるーく飾らないのが好きo(^_^)o
 
 
この2日間は、そんなギャップな
私が色々見れたかなー？？
 
 
 
お誕生日の近いメンバーの、生誕祭もありました。
かずみん、ちーちゃん、あやね、みなみ、おめでとー⊂( ˆoˆ )⊃
 
 
まりかの生誕祭で、お手紙を
読ませて頂いたよ。
 
私の生誕祭のときは、まりかがお手紙を読んでくれたんだ＾＾
 
 
大好き、まりか♡
 
うぇいうぇいうぇい！
 
 
これね、ただの茶封筒じゃないんだよっ
 
ミニチュアなの！切手も 超ミニサイズ(  ´^_^` )
 
エアメール封筒とも迷ったけど、
渋さが可愛いなーって思って、文房具屋さんで買ってきたんです ＊＊
 
 
まりかが言う通り、私はちょっと渋いものが好き( ^ω^ )
 
 
 
まりか推しの皆さん、
まりか生誕祭に参加させてくれて
有難うございました  ＋＋
 
 
  ー
 
 
のぎ天、バレンタイン回は見てくれたかなー( o^^o )？
 
 
私が作ったお菓子はコレっ ＊
 
カラフルな
マカロン、クッキー、ワッフル...
 
に見える、チョコレート です(　ˆoˆ　)！
 
 
中身も見た目も美味しいものを作りたいなーって思ってね！
メンバーからも 好評で嬉しかったです(#^.^#)
 
お料理作るの好きだし、初めてサークルに参加できて楽しかったです ♪
 
 
 
井上小百合 小動物ワールド 、炸裂してましたね（笑）
ちょこまか(･ω･｡≡｡･ω･)ちょこまか 
 
 
私のお料理姿、、
 
「一人、プラモデル作ってるみたい」
 
って、おいい（笑）
 
 
ただ、せっせと動いてるだけでも
冬眠前のリス とか言われるの...( ･ิω･ิ)
 
なんでぇぇー笑
 
 
 
 
あっ前にモバメでも、ふとしたときに
ひめたんと手を繋いでる写真送ったことあったね（笑）
 
私、人見知りだけど、慣れるとすごい懐くタイプ。(　・´ｰ・｀)
 
 
あの時、ひめたんの膝に手を置いてたら、ひめたんがその手を握ってきてくれたんだ( //^o^// )
 
ひめたんは、温泉が苦手らしいけど
私とまりかとだけは、一緒に入ってくれる！
 
 
 
乃木坂46SHOW! 見てくれて有難う。
今回、コント初登場で台詞はちょっとだったけど、参加できてすごく楽しかったです！
また出れたらいいなあー
 
 
今日の録音会、似顔絵会も
有難うございました(*^_^*)
 
では！！！
 
 
 
 
 
                     さゆヽ(｡･ω･｡)正義
