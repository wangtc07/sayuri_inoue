2015/1030Fri井上小百合ヽ(｡･ω･｡)かざぐるま

 
 
 
最近、もの忘れがひどくて
我ながら困っています(´=ω=｀)
 
日傘と、時計と、リップを
3日間連続で失くしました(  ;  ;  )
 
どれも買ったばかりで
気に入っていたやつ...｡･ﾟ(ﾟ⊃ω⊂ﾟ)ﾟ･｡泣けるぜ
 
 
あんまりにひどいんで、最近は
手帳と携帯のアプリのWで、
やることなすこと全部
メモするようにしています。（笑）
 
 
 
そういえば、結構前のブログで、
(    夏休み中、富山の他にも
色んなところに行ったので
次のブログで書くね〜   )
とか言ったのですが
すっかり忘れていました！( ๐_๐)〣
 
 
ごめんなさい
 
 
 
お母さんと、色々
旅行に行ってきたよ♪
 
今回は、福島県の会津と、
栃木県の那須に行ってきました❀
 
 
 
 
このブログを読んでくださってるみんなへ、
沢山写真を撮ってきたよっ(﹡ˆˆ﹡)！
 
 
 
まずは、、
 
『塔のへつり』
昔ここは全て海だったんだそうです。
吊橋を渡って入った洞窟の奥に、
虚空像菩薩が祀られていました。
 
 
 
『大内宿』
江戸時代の町並みを残す宿場町です。
キレイな花が咲いていました。
 
 
 
用水路では、野菜や飲み物を冷やしていました。
少し水に触れてみたら、びっくりするくらい冷たかったです。(*°_°*)
 
 
 
 
どうかな、少しでも伝わったなあ？○゜
 
 
風情があってとっても素敵でした。
心が豊かになれた気がしました。
 
 
 
乳製品が好きなので、
チーズガーデンってとこにも寄って、
色々買ってきたので、
料理するのが楽しみです✡
 
 
 
あと、最近会えてなかった
おばあちゃんにもお土産を買って、
旅行のあと、おばあちゃんのとこへ
渡しに行きました。
 
 
詳細は...  モバメで送りますっ！
 
 
載せきれなかった写真たちも、
またモバメでいっぱい送るねっ(＊´ω`＊)
 
 
キレイな景色と一緒に、動画も
お母さんに撮ってもらったので、
それは755にアップします( ﹡ˆ_ˆ﹡ )♭
 
 
 
お母さんもおばあちゃんも
有難う。
 
楽しい旅でした
 
また、紅葉も見に行きたいなっ
 
 
 
  ー
 
 
 
先日、日本武道館へ
サカナクションさんのライブを
観に行ってきました！( ﹡ˆoˆ﹡ )
 
 
お仕事が終わったあとダッシュして
なんとか途中から観ることができました。( > <。)
 
本当に素晴らしかったです！
沢山、沢山、刺激を受けました！
 
私も頑張るぞー♪
 
 
 
  ー
 
 
お知らせ。
 
 
週刊プレイボーイさんの、
『乃木坂46物語』
 
2週連続で井上をピックアップして頂いて、
本当に嬉しいです。有難うございますo(_ _*)o
 
 
泣きながら記事を読みました。笑
取材のときも何度も泣きそうになったけど、
やっぱり思い出すと...
 
頑張らなきゃなって思います。
 
 
絶対絶対読んで欲しいです！
 
 
 
 
明日31日、日本武道館で行なわれる 
「日テレ HALLOWEEN LIVE 2015」
に、出演させて頂きます。
よろしくお願いします(*^_^*)！
 
 
 
 
そして同じく明日、17:00〜
ニコニコ生放送「乃木坂46～メンバー生出演＆代々木ライブ初フル配信～｢今、話したい誰かがいる｣発売記念！SP特番」
 
まなつ、生駒ちゃん、みさ先輩、
若さま、私の、5人で
生出演させて頂きます。
http://live.nicovideo.jp/watch/lv239941752
 
よろしくお願いします\(﹡ˆᴗˆ﹡)/
 
 
 
では！！！
 
 
 
 
                     さゆヽ(｡･ω･｡)正義
