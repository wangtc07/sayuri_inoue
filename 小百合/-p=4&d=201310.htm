2013/1007Mon井上小百合ヽ(｡･ω･｡) 力を くだーさゆにゃん

 
 
 
乃木坂って、どこ？見てくださった方
有難うございます(* ^o^ *)！
 
 
センター堀ちゃん。
どんな風になるのか、まだまだ全然誰にもわからないけど、確実に乃木坂は変わる。
一歩踏み出すのってすっごく怖いと思うけど
みんなで、一歩ずつ進んでいけば大丈夫 ＊＊  がんばろう ( 〃^o^ 〃 )
わからないことがあったら、何でも教えるから、支えになれるかわからないけど、いつでも相談してね ♩♩
 
 
新しく選抜に入ったメンバー、残ったメンバー！
本当におめでとう。
とくに、初選抜の、ひなちゃん、みさ先輩、ひめたんは、同じ上京組みで、
色々話したこともあったから、自分のことのように嬉しくて喜んでしまった ( ;  ; )
選抜になっても、きっと辛いことも沢山あると思う。
そういうときは、遠慮なく頼ってほしい。みんなで助け合って高め合って、乃木坂を良いものにしていこう ＊
 
 
 
 
そして、改めてご報告。
井上小百合は、アンダーメンバーになりました。
 
発表が終って一番最初に思い浮かんだのは、応援してくださるみなさんのこと。
きっと何て言っていいかわからないっていう気持ちの方もいると思いますが...
 
私は落ち込んでいません！
こんなことで弱る人間ではない。
 
ここで泣いたり落ち込んだりしたら
自分に負ける気がして。
だから、より一層やる気が芽生えています！ヽ(｀･ω･´)ﾉめらめら
 
 
今は、きっと我慢の時期なんだと思います。
いつか光がさすことを信じて、
直向きに、どんなことでも、頑張っていくだけです。
 
 
今年のプリンシパルも成績残して、
ダンス未経験から始まって、努力と根性で ダンス七福神に選ばれて、
(未経験者で選ばれたのは、私 一人)
歌も努力して、それが実って歌番組などでワンフレーズだけだけど ソロで歌わせて頂いたり、何度かフロントも務めさせて頂きました。
ただ頑張る。だけじゃなく、ちゃんと結果も残してきた。
まだ努力が足りないのでしょうか...
今まで全てに選抜入りしていた人の中で、周りは全員福神を経験して、自分だけ最後列の景色しか見たことがなかった。
一度でいいから、あと少し、もう少しだけ、みなさんの近くで歌を、歌ってみたかった。。
 
でも、アンダーとして、これからきっと色んな景色が見えるだろうし、色んなことが学べると思います。
そういう機会を与えて頂いて、感謝しています＾＾
 
ほら、ヒーローだってピンチを経験したときに強くなるし！
何もない、ただまっすぐな道ではつまらないし、
そういう物語をつくって頂いたんだと考えてます！ポジティヴセイ
 
ピンチはチャンスなのだっっっ＼( ^o^ )／
 
 
 
応援してくれる優しい家族や友達、支えて下さるスタッフのみなさん、かけがえのないメンバー、
そして、サユリストのみなさん。
 
こんなに大切な人が 沢山いて、、
私は、しあわせです。
 
有難うっていつも感謝しています。
本当に本当に、有難う。
 
みんなのことが、大大大好きっ。
 
 
だから、絶対、
頑張るんだっ ( 〃 ˆ ˆ 〃 )！！
どうか、力をください。  どうか...
 
私には、夢がある。目標がある。
 
 
今まで、たくさん、悔し涙を流してきました。
 
 
次に流すのは嬉し涙です。
私は笑顔で言いたい。
 
 
 
 
 
 
 
 
 
 
 
 
絶対にこのグループのトップになります(* ^_^ *)！
 
 
 
さゆにゃんヽ(｡･ω･｡)正義！
