2014/1012Sun井上小百合ヽ(。・ω ・。)見 てね〜

 
 
うにゃ！
ちーちゃんチリチリ頭！（笑）
 
 
本日の乃木どこで、アンダー曲
「あの日 僕は咄嗟に嘘をついた」
のスタジオライブが放送されますっ(*^o^*)多分
 
 
歌衣装、乃木坂らしくてとってもカワイイよ♪
 
ダンスも、激しめのダンスに挑戦してます。
 
前回のアンダー曲「ここにいる理由」と同じ振付師さんで、
そのときよりパワーアップした振りを付けて下さいました♪
 
お楽しみに〜(  *^o^*  )
 
 
 ー
 
 
のぎ天！
 
 
みんなで、クッキーの指輪して
お菓子のお家みたいに記念写真したよっ*\(^o^)/*
 
ここのお店、すんごく美味しかったし可愛かった！またみんなで行きたいなーっ
 
あと、大好きな漫画を久々に描いたりして楽しかったです♪♪
 
絵描くの好きだけど、4コマリレーってのは初めてだったから面白かったな。
 
ぜひ、見てみてねー(*´ο`*)
 
 
 ー
 
質問にこたえるよ(ミ＾o＾ミ)

◎  ひめたんがおでこ出してたけど...さゆニャンは出すのと出さないの、どっちが好き？
 
＊  おデコ、狭くてお母さん似で
眉毛はお父さん似で...
恥ずかしいから、あんまり出したくないなあ(^ω^;)笑

◎  さゆちゃんは時間あったら免許とりたいとか思う？
 
＊  とりたいけど...普段ボケーッとしてるし、フワフワし過ぎてて
運転とか絶対危なそう（笑）
 
どうせお兄ちゃん(←オタクの)が
運転してくれるからいいや〜っ(*´ο`*)ごめんね
まだ甘えさせてね。笑
 
 
 
▼  Name :  尼崎のゆっきー さん
 
◎さゆ、自分の長所と短所を
答えて下さいって言われたら、
さゆは何て答えるのかな？
 
＊  短所なんていっぱいあって書ききれないよ(´;ω;`)
日々努力です...
 
いくつかあげるなら、
いい意味でも、悪い意味でも
のほほんとしてるトコ。
自分の意志を最後まで貫くトコ。
 
 
 
▼  Name :  けんちゃん  さん

＊  トランペット！
金管はカッコイイよね！
あと、テナーじゃなくてアルトのほうもやってみたかったな。
 
 
  ー
 
 
明日もアンダーライブ2公演！
みんなで楽しもうね(  *^_^*   )
 
くふっ
 
 
では！！！
