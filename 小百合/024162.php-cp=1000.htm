2015/0619Fri井上小百合ヽ(｡･ω･｡)今日！です

 
 
 
えーっと、
お話したいことが沢山あります(´,,･ω･,,`)
 
 
まずは...
 
本日発売の、『ヤングガンガン』さんで
巻末グラビアをさせて頂きました！
 
一人の撮影は、嬉しくもあり、ちょっぴり緊張。
 
でも、撮影後にスタッフの皆さんと食べたお寿司が
とっても美味しかったです！！
 
最近、まぐろが食べれるようになったので、得意気にまぐろを沢山食べました(﹡ˆωˆ﹡)
 
 
無邪気ないつもの私だったり、
 
 
 
 
わーい
 
 
 
ライブのときに見せるような
スイッチオンの私だったり、
 
色んな表情が見れると思います。
 
生駒ちゃんと私のクリアファイルも付いてくるので、
ぜひぜひ、チェックして下さいね ♪♪
 
 
 
  ー
 
 
「のぎ天」  台湾クイズ
見てくれたかなー？？
 
もうネタバレしちゃっていいのかな？
見たっていう前提でお話しちゃうと...
 
 
なんと、井上、
優勝しました〜\( *´ω`* )/パンパカパーン
 
最近いろいろ優勝する...( 'ω'  ;)恐縮
 
 
頂いたプレゼントは、有難く
みんなに配って山分けしました（笑）
 
まだ、忙しくてお食事券は使えてないので、落ち着いたら
メンバーみんなを誘って食べに行きたいです！
 
小籠包っ小籠包っo(,,^_^,,)o
 
 
 
  ー
 
 
先日の、名古屋での
握手会、有難うございました！
 
沢山の方が来てくれて、本当に嬉しかったです。
 
やっぱりこう直接みんなの顔が見れて、「有難う」とか「頑張ろう」とか言ってもらえると、
というか、くだらない会話だけでも、
とってもとっても嬉しいのです。
 
 
ブログのコメントとか、755とかも、
時間があればずーっと見てます。
 
いつももらってばかりで申し訳ないけど...
 
 
 
 
1部は、ガウチョ！
 
 
 
2部、3部は、
妄想リクエストのときの、ワンピースでした
 
 
実はこの服、ずーっと前の握手会でも、着たことがあるんですよー○゜
 
あれはいつだったけな...(*´ー`*)うーん
 
忘れちった。笑
 
 
まあ、物持ちがとてもいいんです(﹡ˆ﹀ˆ﹡)
 
 
 
 
  ー
 
 
現在発売中の週刊ヤングマガジンさんに、
じょしらくの、チーム「く」で
グラビアをさせて頂いてます
 
 
役をちょこっとイメージしたお着物を着させて頂いてるんですが、
 
 
＼じゃーんっ♪／
 
これは、ヤングマガジンさんだけの
特別なお衣装です！
実際の舞台では着ないよ〜
 
 
それに、お着物をよく見ると、、
何かに気付きませんか...？？(  *ω*  )
 
 
そう、猫ちゃんが居るんです！
 
さゆ、にゃん、、、
 
（笑）（笑）
 
 
 
 
そしてそして、！！！
 
 
チーム「く」は、いよいよ
今日が   初日です！！！！
 
実は、このチームはそれぞれがすごく忙しくて、全員揃ってお稽古できたことが他のチームより圧倒的に少ないんです。
 
自分も、出演させて頂くもう一つの舞台「帝一の國」との両立で、目まぐるしく
ボーっとしてしまうことが多くて（笑）
( いつもボーっとしてるけど笑 )
 
 
でも、他のチームのみんなが支えてくれたり、演出の川尻さんはじめ、沢山のスタッフの皆さんと、
落語の先生や、ダンスの先生のおかげでここまでこれたので、
 
感謝の気持ちも込めて、
素敵な初日を迎えられるように、
頑張りますっ♪
 
 
 
前に、グラビアザテレビジョンさんのインタビューで私が話していたこと。
 
「大好きなお芝居を通して、グループに貢献したい」
 
ということは、まさに今が
自分にとって、とても貴重な時間なので、頑張りたいです。
 
 
 
創作落語、チームのトップバッターは私が務めます( >_< 。 )
 
お芝居は好きだけど、落語は初めて。
 
楽しんで笑ってほしいなー！気楽に観てね♡
よろしくです( *ˆoˆ* )
 
 
 
 ー
 
追記
 
 
学欄歌劇「帝一の國」
 
海帝高校だより(公式ブログ)が毎日アップされています。
 
お稽古の風景が色々見れると思います。
 
http://yaplog.jp/teiichi/
 
いつかは私の番も回ってくるのでお楽しみに♪
 
 
 
 
                     さゆヽ(｡･ω･｡)正義
