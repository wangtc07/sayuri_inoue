2015/0115Thu井上小百合ヽ(｡･ω･｡)白目

 
 
1月14日　TVガイド（生田、井上、松村）
1月27日　Cool-Up Girls（井上、桜井、中田、永島、能條）＊表紙
1月24日　SamuraiELO(井上、優里、高山、永島、橋本)
1月29日　ブブカ(井上、優里、永島)
 
 
載ってます！見てください(*^o^*)
 
 
Cool-Up Girlsさんは、94年組で
表紙も飾らさせて頂いてます！
 
 
SamuraiELOさんは、白いふわふわの
衣装に高めのツインしたよっ！
楽しいメンバーでした＾＾♪
 
 
  ー
 
 
 
只今、乃木坂メンバーが全国に散らばって
ファーストアルバム「透明な色」をPRするキャンペーンを行っています！
 
 
私は、北海道に行ってきましたあ！！
 
でっかいどー！どーーーんっ
 
 
北海道チームは、まいやん、ゆったん、私の3人(*^_^*)
 
雪を見てはしゃぎ、時計台の前を通ってはしゃぎ、美味しいものを食べてはしゃぎ...
まるで修学旅行生かのように3人で
はしゃいでました（笑）楽しかったなー～(＾Д＾～)
 
北海道新聞のキャラクター、
ぶんちゃんのマネをしてパシャリ。
ピース笑
 
 
ぶんちゃんの足下の倒れこんでるやつが気になり過ぎて
ピンショット撮っといたよ。
 
 
 
 
2日間で、新聞、テレビ、ラジオ、イベントと
沢山 出させて頂きました！
 
 
北海道の皆さんは、放送や掲載をお楽しみに！
 
 
 
 
 
 
 
 
 
クイズで貰った大好物のクッキーも、すぐさま食べちゃいました。笑
 
 
熱烈！ホットサンド！にも、コメント出演させて頂いたのですが、(3月放送予定)
 
個人的に、ものすごくサンドウィッチマンさんが好きなので、嬉しかったです(*^^*)
 
いつかお会いできたらいいな～
 
 
 
ラジオの公開収録では、約2000人のお客さんが来て下さり、感動しました( ; ; )
寒い中、ほんとに有難う＊＊
 
この写真を見せて、こんなに沢山の方が来て下さったから、
全国ツアーで北海道もまわってあげたいです( > <    )
 
ってスタッフさんに、お願いしておきましたっ！笑
 
 
絶対絶対、また会いに来たいですo(^_^)o
 
 
北海道の皆さん、本当に
有難うございました！！！
 
 
今回は行くことができなかったけど、他の地方の皆さんも
ラジオなど聞いて下さって有難うございました(  *^o^*  )！
 
 
 
 ー
 
質問にこたえるよ(ミ＾o＾ミ)
 
 
 お名前  「 なこ＊♡ さん 」
 
◎ ハタチになって何か変わったと思うことはありますか？？
 
＊  うーん、小さい頃ってハタチってもっと大人な気がしてたんですけど...
何にも変わってないです（笑）
 
残念ながら描いてた大人とは、
自分はかけ離れてるみたい（笑）
 
 
 
お名前   「 おじゅん  さん 」
 
◎  お寿司は何が好き？
サビ抜き？
 
＊  こう見えて？ わさび大好きなんですo(^_^)o
たっぷりわさびをつけて、ツーンってするのが好き！
お寿司は、たまごとエビがすきだけど、
最近、まぐろが食べれるようになりました。
 
 
 
  ー
 
 
最近、ピスタチオさんにハマってます。
上手にマネ出来るようになりたくて、白目の練習頑張ってます（笑）
 
 
「 なんのっ！？(白目) 」
 
 
では！！
 
 
さゆヽ(｡･ω･｡)正義！
