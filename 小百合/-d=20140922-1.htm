2011/1215Thu井上小百合ヽ(｡･ω･｡)伝えたいこと

１２月１４日。

今日は、わたしが生まれて、
１７年目の記念日＊



私の父は、無口であまり感情を
表に出さない人です。

私が芸能界に入ると言ったときも
あまり賛成してくれなくて、

そのままお互い何も言えずに
私は家を出てしまって、
すごく後悔してたんですが、

実はテレビとか見て、
私のこと応援してくれてるって
お母さんから聞いて、

お父さん、何も言って
くれなかったから…

驚いて、嬉しくて涙が出ました。


お父さん、あのときは
何も言えなくてごめんなさい。
私、お父さんのこと大好きです。

１７年間、有難う（ ;  ; ）

母は、厳しくて努力家な人です。

沢山迷惑かけて、
ごめんなさい。

一人暮らしを始めて、
初めて母の大変さがわかりました。

５人兄妹で、初めて出来た女の子
だったから、私が生まれたとき
お母さん、バンザイして
喜んでくれたんだよね。

そんな私が１６歳で親元離れて、
初めて涙を流す母を見たとき、
私はこの涙を無駄にしては
いけないと誓いました。

大好きなお母さん、
私を産んでくれて、
誰よりも愛してくれて有難う。


それから、地元のお友達。

朝からクラスのみんなで
おめでとうって電話してきてくれて、
泣かない訳がないです（ ;  ; ）

もう、あの教室には私は居ないのに
黒板にこんなふうに書いて
私のこと、祝ってくれて
嬉しかったです。

私は、いい友達を持って
幸せものです。

本当に有難う（ ;  ; ）


本当は、みんなと修学旅行
行きたかったし、
もっと勉強もしたかったし、
部活も頑張りたかったし、
一緒に卒業したかった。


両立も考えました。

でも、それじゃあ、

そんな中途半端な状態で
乃木坂をやっても、
ファンの皆さんや
メンバーに失礼だから。

私は、色んなものを捨てて
乃木坂に人生をかけて来たんです。

だから、私は、
もっと頑張らなきゃ
ダメなんです。

正直、今の自分には
満足できません。

悔しいことばかりです。

こんな悔しい誕生日、
２度と送りたくないです！


だから、来年の誕生日は、
すっきりした気持ちで
迎えられるよう、１７歳のこの
１年間、精一杯頑張ります。


最後にファンの皆さん、
沢山のおめでとうコメントや
お手紙、誕生日プレゼント、
すごく嬉しいです（ ;  ; ）
本当に有難う御座います。

感謝しきれないほど、
感謝しています。

苦しいときも、
皆さんの優しさに触れる度、
元気や勇気を頂けて、

皆さんのおかげで、私は
どれほど強くなれたか。
皆さんにとっても、私が
そういう存在であれますよう、
頑張ります。

と、泣きながら記事を書いていたら
何時の間にか誕生日、
過ぎちゃってました(笑)

ごめんなさい(´･ω･`)(´-ω-`))ぺこり


メンバーのみんなに対しても、
感謝があり過ぎて書ききらないので、
また後日、書かせて頂きます。

では、おやすみなさい○°

いい夢見てねっ(o^^o)
