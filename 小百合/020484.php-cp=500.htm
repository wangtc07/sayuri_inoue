2014/0924Wed井上小百合ヽ(。・ω ・。)袴 着たよー

 
 
こんばそわーヽ(^ω^)ノ
 
 
昨日は、お茶会とカルタ大会、
2つのイベントがありました。
来てくださった皆さん当選おめでとう！！
 
有難うございましたっ＊＊
 
 
お茶会のトランプは、
負けず嫌いだからガチ勝負！（笑）（笑）
前回に続き、1位でした(｀・ω・´)キリッ
同率だったけど笑   楽しかったなあ！
 
 
 
カルタ大会では、袴を着ました。
 
着れてほんと嬉しかったあ(* ^o^ *)！！！！
 
日本て素敵だ...  ／(^o^)＼おういえあ！
 
 
94年組と、「4のポーズ」！
 
ひとつ上のゆったんは、93年の「3のポーズ」！
 
 
 
1994.12.14 生まれコンビ！
 
 
 
さゆったんコンビ！
 
 
 
 ー
 
質問にこたえるよ(ミ＾o＾ミ)
 
 
▼  Name :  ぽんちゃん さん
 
◎  ひめたんとかなりんの二の腕はどちらがお好みですか？
 
＊  うーん、甲乙つけがたいな...(。-`ω´-)☆
 
ひめたんは、むきゅむきゅ。
かなりんは、もにゅもにゅ。
と言ったところか...(*´ο`*)
 
 
 
▼  Name :  ヒサ!Bicycle さん
 
◎  さゆに質問なんやけど、もし乃木坂の曲を吹奏楽で演奏するなら何がいい？
 
＊  月の大きさとか、金管かっこ良さそうだなー！
 
 
 
▼  Name :  ニャンいこ派   さん
 
◎  さゆニャンは筋肉痛になったら何日ぐらい続く？てか、さゆニャンって筋肉あるの？
 
＊  うーん3日間くらいかなぁ〜
そりゃあ、あるよー！（笑）
 
 
 
▼  Name :  tom7793 さん
 
◎  (お料理)可愛い容器に入ってるけど市販されてるの？
 
＊  そうだよー♪
 
お買い物に行って、
かわいい食器とかを選ぶのも楽しい(*^o^*)○゜
 
こないだ、お花型のお皿を買ってきたので、使うのが楽しみですっ♪
 
 
 
▼  Name :  みーあ  さん
 
◎  (パンケーキ)やっぱり食べるのに数時間かけて食べるの？
 
＊  あのパンケーキ、ひめたんと2人でも食べきれなくって
結局、お持ち帰りにしてもらったの＾＾;（笑）
 
 
 
▼  Name :  にくきゅうくんくん丸 さん
 
◎  かなりんブログに、「袴の色は自分で選んだ人と自分じゃない人いるけど」ってあったけど、さゆちゃんはどっちですか？
 
＊  「さゆは、何でもいいよー(*^^*)」って言って、みんなが選んだあと、最後に余ったやつを着ました（笑）
でも、黄色の袴キレイだったし良かったです＊＊
 
 
   ー
 
 
明日は、早起きでお仕事！
 
自分の夢に一歩でも近づけるものなので、頑張ってきまーすｐ(*^-^*)ｑ
 
 
では！！！
 
 
 
さゆヽ(。・ω・。)正義！
