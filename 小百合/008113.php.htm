2012/1028Sun井上小百合恥ずかしかった（笑）＋＋

 
 
前回の誰でしょうかクイズの答えは、
 
 
 
 
 
 
 
... デデデデデデデ！
( ドラムロール )
 
 
 
 
生駒ちゃんとまいまいでした～＼(^o^)／ぱんぱかぱーん！
 
 
どおだったかなぁ～？( *^_^* )
 
 
 
 
そして！
本日は、横浜にて個別握手会でした！
 
起こしくださった皆さまっ ^o^
ありがとうございました！仏ビーム！
 
 
今回は、ハロウィンということで
メンバーそれぞれが、サプライズで
いろいろ仮装したりしていました☆☆
 
かなりんは、マジすかのサドコスしてましたね！！！
もう、わたし、麻里子さまダイスキなんです＞_＜
めっちゃ感動したぁ～（ ;  ; ）
もうずっとかなりん見てカッコイイ連呼してましたっ（笑）
 
カッコイイ女の子だいすき（＾ω＾）
 
 
 
そんなわたくしは、
イベントでの懐かしきプリキュアを...(*_*)
 
 
キュアリズムさまです！正義っ！＼(^o^)／
 
二次元は素晴らしい＼(^o^)／きらぴかっ
 
実は以前、ライヴイベントでキュアメロディさまのコスをして
めっちゃ恥ずかしかったからもうやらない！って言ってたんですが、
今回は、ハロウィンってことで特別に
いのうえ、頑張りました...(^_^;)
決して毎回こういう格好をしてるとかコスプレが趣味とかではないですっ（笑）
初めてきてくださった方はびっくりでしたよね...^_^;ワラ
 
 
でも、やっぱりめちゃくちゃ恥ずかしくて恥ずかしくて耐えきれず（笑）
5部の前で、見せる予定のなかった
今日普通に着てきた私服に
着替えちゃいました（笑）（笑）
 
 
 
だってホントに恥ずかしかったんだもん～！（笑）
 
二次元は二次元だからこそ素晴らしいのです。
 
またこれでもうコスは当分しません...
からねっ...(*_*)
今日見れた人はラッキーです！（＾Ｏ＾）
 
 
ちなみに私服は、WONDERROCKETの！
実は、生駒ちゃんと色違いのお揃いなの＞_＜
 
 
それから、ゆみ姉、今日はお疲れさま(*^_^*)
皆さんも、本当にありがとうございましたm(_ _)m
 
 
とにかく今日も、すごくすごく楽しかったです(*^o^*)
 
感謝することばっかりで、ごめんねだけど...
 
本当にありがとう
 
 
 
おやすみなさゆにゃん ♪♪
 
お耳 てへぺろ ////
 
 
 
 
 
 
 
さゆにゃんヽ(｡･ω･｡)
