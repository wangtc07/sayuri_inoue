2015/1018Sun井上小百合ヽ(｡･ω･｡)次の目的地を 描くんだ

 
 
 
色々、整理がつかなくて
遅くなっちゃってごめんなさい！
 
 
 
「すべての犬は天国へ行く」
無事、全公演終了致しました( ﹡ˆˆ﹡ )
 
わー
 
グループにとって、新しい挑戦
となったこの舞台に、自分が
関われたことを有難く思います。m( __ )m
 
 
すごくすごく葛藤しましたが、
その分、グループとしても個人としても、
成長できたような気がします。
 
 
今までの乃木坂46にない、
新しいことに一歩踏み出すのは
とても怖かったですが、
楽しくもありました。
 
 
 
ご観劇下さった皆さま、
有難うございました。
 
 
メンバーの皆や関係者の方々も沢山、
忙しい中来て下さって有難かったです。
 
 
なにより、お客さんには
沢山支えて頂きました。
 
稽古の時は不安しかなかったけど、
公演が始まると色んなリアクションを
お客さんが下さるので、
毎回発見があって楽しかったです。
 
私が演じたエルザという人物は、
外部からの流れ者で、
唯一物語に対して客観的なので
 
常にお客さん目線で、舞台全体を見て、
それによってその場で芝居を
変えたり。ニュアンスを工夫したり。
 
 
難しかったですが、実践的に
とても学べました。
 
 
 
説明的な台詞やツッコミが
多い点でも苦労しました。
周りのスピードや空気感に合わせて、
感情だけでガンガン行けないところ。
 
 
客観かと思えば、主観に
陥ってるときもあったり、
心の揺れ動きを表現するのも
難しかったです。
 
 
 
今まで演じたことのある役柄とも
全く違うし、細かい仕草まで
全部自分と真逆の役づくりに、
沢山葛藤しました。
 
 
 
だけど、エルザに出会えたことは
すごくすごく大きかったです。
 
 
両親をなくし、23歳という若さで
一生を終えた、とても悲しい人。
それでも彼女はいつでも明るく
振る舞って、強く見せてた。
笑いながら死んだ彼女の人生が
なんだか深すぎて深すぎて
私が背負うには重た過ぎたかもしれない。
 
本当はとても弱くて、優しい人。
 
 
 
エルザを演じることは、
とてもプレッシャーでしたが、
エルザを誰よりも愛してるという
自信はありました。
 
 
いつの間にか、自分の中に
エルザが存在しているような
感覚になってました。
 
この役に出会えて良かったです。
有難うございました✡
 
 
 
 
 
そしてそして〜♪
 
 
ぜひ、白鳥美美子ちゃんのほうも、
沢山の方に見てほしいです( ﹡ˆ_ˆ﹡ )
 
 
 「【第二章】學蘭歌劇『帝一の國』－決戦のマイムマイム－」
12月18日、DVD発売します。
 
只今絶賛予約受付中です✡
 
 
歌劇なのですが、今回は美美子ちゃんの
ソロ曲までつくって頂き、
一人で歌うシーンもあります。
 
 
「エルザ」も「美美子」も
どっちも「井上小百合」であって、
どっちも違う人。
 
 
もっと色んな役をやってみたいなっ
 
 
個々が実力をつけることで、もっと
大きな集団になれると思うので
 
乃木坂46へ、どんどん新しい風を
送っていける存在になれるように、
私らしく頑張りたいです( ﹡ˆ_ˆ﹡ )
 
 
 
 
 
遅くなりましたが、前回のブログに
まだ書ききれてなかった
カンパニーの皆さん！
 
 
谷松さん
私がお仕事で稽古に出れない日は、エルザの代役でずっと入って下さってて、私の分まで稽古して下さいました。
とってもとっても、お世話になりました。本当に有難うございました( > <。)
 
 
榎本さん
実は、帝一の國の第一章も第二章も観に来てたらしくて、その時は、可愛い子出てるなーって思って下さってたんだとか(  ;  ;  )泣
共演できて良かったです！
ずっとギューしてくれて嬉しかったです。照
 
 
山下さん
一度、たまたま駅でばったり会って
稽古場まで一緒に歩いたことがありました。色んなお話をしましたが、
すごく面白くて、もっともっとお話してみたかったです。
色んな引き出しがあるって褒めてくださって、嬉しかったです。
 
 
伊藤さん
役のメイクが面白すぎて、日に日に濃くなっていってるのがもうとんでもなく面白かったです（笑）
あんまり沢山お話できる機会がなかったのが残念...もっと沢山お話してみたかったな(　ˇˇ。)
 
 
山田さん
特撮話ができてほんとに嬉しかったです！めっちゃノリが良くて面白くて、あと、可愛いって言ってるキャラクターが、みんなきもかったです。（笑）（笑）
 
 
鳥居さん
楽屋での会話が面白すぎて面白すぎて、毎日ほんとにもう笑いが止まらなかったです。
とくに猫背さんとの会話が、、（笑）
でも、私が毒虫に刺されて帰ってきたとき、一番心配してくれました。
 
 
東風さん
めちゃくちゃ美人なのに、とってもチャーミングで、素敵すぎる方。
何度も顔に水を噴いて、本当に申し訳ありませんでした。笑
「お芝居だから仕方ないよ〜」って優しくて嬉しかったですが、最初の頃はもう稽古に行きたくないくらいそのシーンが苦しかったです（笑）
 
 
 音さん
実は同い年ってことに、結構経ってから気付きました（笑）でもそれからもなんとなくお姉ちゃんみたいに接しちゃいました（笑）
音さんも妹みたいに接してくれました（笑）芝居の中で何度も殺してごめんねっ( ﹡ˆoˆ﹡ )笑
 
 
 
 
私のほっぺをいつもムギュムギュ
してくれた、甚古さん
えへ〜
 
人見知りだけど、慣れると懐っこくなるタイプなので、
こういうのすんごく喜んでしまいます(照)
 
 
 
 
そして、犬組のメンバーみんな！
 
 
このメンバーと舞台に立てて
本当に良かったです❀
 
この期間、それぞれ葛藤しながらも、
グループとしての将来をすごく
考えていたと思います。
 
舞台が終わってからも、また8人で
ご飯行こうねって計画立ててます(﹡ˆᴗˆ﹡)
 
とっても絆が深まりました。
 
みんな有難う(*^_^*)♪
 
 
 ー
 
 
「今、話したい誰かがいる」
MVオフショット
 
 
休憩中は、大自然の中で遊びまわってました(﹡ˆˆ﹡)
 
 
蜘蛛の巣のすみかを見つける。
「わーー」
 
 
 
 
蜘蛛の巣に引っかかった虫を助けてる私。「大丈夫か〜」
 
尚、まりかと生駒ちゃんはまだ蜘蛛の巣に夢中。
 
「？？」のみなみ。
 
 
 
 
 
もう意味のわからない遊びを始めてる。（笑）
 
 
 
 
本日は、久々の個別握手会♪
 
お話したいことが沢山たまってる( ﹡ˆoˆ﹡ )
 
恥ずかしいけどコスプレするよっ
ハッピーハロウィン♡♡
 
 
では！！！
 
 
 
 
                     さゆヽ(｡･ω･｡)正義
