2015/0517Sun井上小百合ヽ(｡･ω･｡)乃木中みてね ん

 
 
 
 
ん～
 
 
 
 
れいか、お誕生日おめでとう！
 
乃木坂の最終オーディションの日に
初めてれいかと会話をして、途中まで一緒に帰ったんだけど
その時はすごく大人びて見えて、同い年だとは思わなかったんです。
 
でも今は、日に日に、絡みがうざくなってきてます。（笑）
まぁそんなとこも好きです。笑
歌とダンスはれいかを尊敬してますo(*^_^*)o
 
 
 
そして、日村さん
お誕生日おめでとうございます！
 
乃木坂工事中に変わってからも、バナナマンさんがMCをして下さって、
本当に嬉しいです。
これからも、公式お兄ちゃんとして
私たちをよろしくお願いします( ﹡ˆ ˆ﹡ )
 
 
 
そして、今日の「乃木坂工事中」は、
松井玲奈さんとの最後の回になります。
 
兼任ということになったとき、最初はやっぱりとても不安だったけど、
乃木坂46に来てくださったのが、
玲奈さんで本当に良かったです。
 
ということで、今日の放送、
良かったら見てください( ﹡ˆ ˆ﹡ )
 
よろしくお願いします
 
 
 
  ー
 
 
質問にこたえるよ(ミ＾o＾ミ)
 
 
▼  おなまえ: ニッキーほんた⊿ さん
     ━━━━━━━━━━━━
 
 
◎  何だかサユリストになって、ここのブログに来るのも楽しくなってきたなー。
さゆにゃんの推し歴が浅いから分からない事ばかりなんだけど、ブログコメントはちゃんと読んでくれますか？
 
＊   読んでますよー( ﹡ˆ ˆ﹡ )
 
寝る前とかに、携帯を開いて毎日読んでますっ
おんなじのも、何度も読み返しちゃったりしてます。
 
皆さん、いつも有難う
 
 
  ー
 
 
この時期って、
素敵な季節ですよねぇ
 
 
初夏のこういう景色って好きです。
 
 
これは埼玉県なんですけど、
いつか大っきいカバン持って、もっと遠くのキラキラした景色を
見に行ってみたいです。(*´ω`*)
 
皆さんは、どっか行きたいとこありますかー？？
 
では！！！
 
 
 
 
                     さゆヽ(｡･ω･｡)正義
