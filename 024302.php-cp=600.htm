2015/0704Sat井上小百合ヽ(｡･ω･｡)どうでもいいけどさ 水筒って便利だ

 
 
 
シュパッ
 
 
アンダーライブくらいからずーっとノンストップが続いてます( *ˆ_ˆ* )！笑
 
舞台2つに、ドラマに、12枚目制作&プロモーションに、取材や撮影、
 
色んなお芝居もさせて頂いて、ここ最近で慌ただしく
5、6人くらいの人間になりました（笑）
 
 
でも、そんな流れに左右されず、マイペースに丁寧にコツコツ、
でもって、ゆるゆると猫のように生きてます(﹡ˆωˆ﹡)元気
 
そんな感じで、隙間のない中で
ちょこまかとブログを書いてたんですが
時間かかっちゃってごめんなさい(  *ω*  )
 
 
 
 
お知らせです！！！！
 
 
＊   本日！
日テレ「THE MUSIC DAY 音楽は太陽だ。Part2 」生放送
━━━━━━━━━━━━━━━━
 
ぜひ見てください( *ˆoˆ* )
 
 
 
＊  日経エンタテインメント！
━━━━━━━━━━━━━
 
まりか、ひめたん、堀ちゃんと
アンダーライブセンター対談をやらせて頂きました。
 
色んなことを細かく振り返っていきました。そして、これからの乃木坂についても考えました。
このメンバーとお話ができて、本当に良かったです。
 
 
 
＊   07/11   23:15～23:45　NHKBSプレミアム
 
『乃木坂46SHOW!  第7弾』
━━━━━━━━━━━━
 
最新曲「太陽ノック」を
フルサイズでTV初披露！
 
さらに、
舞台「じょしらく」の裏側を、みり愛ちゃんがリポートしてくれてます！
 
 
さらにさらに、
まりか、ゆったん、れいか、かなりん、なぁちゃん、若さまとのユニット曲
「他の星から」をテレビ初披露！！
 
この曲を披露できる日がくるなんて、嬉しいよー( >_< 。 )
 
 
 
＊  7/23  『Ｒａｙ』
━━━━━━━━━━
 
まいやんと2人で撮影をしてきました！
こういうファッション誌は私にとって初めての経験だったので、緊張しましたが、楽しかったです！
 
まだ、写真は載せられないので、
発売後にオフショット載せますねꉂ( ˆᴗˆ  )
 
 
 
 
そして、いよいよ、
7月12日からは、學蘭歌劇「帝一の國」が始まります。
 
落語家役が終わって、すぐに今度は昭和の女子高生役。
 
いっぱい頑張ります\( *´ω`* )/
 
 
 
 
それから、ここでちょこっと長くなるんですが、
舞台「じょしらく」のお話をさせてください。
 
 
まずは、一緒に頑張ってくれたチーム「く」のみんなへ。
 
 
かなりん
体調の面でも、本当に苦しかっただろうに、弱音一つ吐かず、
むしろ稽古にあまり参加できなかった私たちのためにいつも頑張ってくれて、フォローに回ってくれました。
自分の弱みは決して見せない、かなりんのそういう、これぞプロのアイドル！ってところ、すごく尊敬してます。
早い段階からキャラ作りも完璧で、チームの土台となってくれました。有難う( ´•̥_•̥` )
 
 
 
 
ことこ
とにかくね、感動しました！
琴子の成長っぷり！！！
最初、プリンシパルで琴子の演技を見たときとはえらい違いでした！
それだけでなく、嬉しいとか悔しいとか楽しいとかもう色んな感情を見せてくれるようになったのが、私としてはすごく嬉しかったです。
楽屋とかでも色んな他愛のない話をしたり、大声で笑い合ったり、、、
 
アンダーライブとかでも琴子の成長をずっと見てきて、親目線で感動しました（笑）
 
 
 
 
かずみん
ムードメーカーになってくれて、いつもみんなを笑顔にさせてくれて有難う。
かずみんが居るだけで、楽屋がパッと明るくなったり、安心感もあったり。
 
ただ、ドラマの撮影で稽古に出れなかったのに、セリフを完璧に覚えてきたときは、
ものすごい頑張り屋さんなんだなって、かずみんのことがもっと大好きになりました。
 
 
 
 
さゆりん
実はこないだ、二人で取材を受けてきました！色んな話ができて、良かったです。
 
この作品を、なぜ乃木坂46が実写化するのか、
その意味を一番表してくれたと思います。
 
この舞台は、
落語家であり、「アイドル」
の舞台でありました。
 
突拍子もないアドリブに笑ったり、泣きそうになったりしました。
 
 
 
 
そして、自分としては...
 
楽しかったですが、
苦しくもありました。
 
舞台経験者だからという声が
どんどんプレッシャーになって、
 
 
安定だからね
 
と言われてしまうと、もう段々よくわからなくなってきて
 
でも、考え込んだ挙句、これは
映像とは違って舞台だということを活かし、
セリフのないときも、仕草や表情などで
空間全体を使って、物語に引き込めるように私なりに努力しました。
 
見る側にわかりやすく世界観をつくりあげることに、工夫をしたり。
 
私の演じたキグちゃんは、かわいい顔して毒を吐く子だったんですが、
裏と表の、表情や声の使い分けにも気を付けました。
 
 
キグちゃんに出会えて良かったです。
この作品に出会えて良かったです。
 
 
千秋楽、私がいつもアドリブで使うおもちゃの中に、
『千秋楽、おめでとう』と書かれた折り紙が隠されていました。
 
それを本番中に見つけたとき、
思わず泣きそうになりました。
必死に堪えましたが( ´•̥_•̥` )
 
期待に応えなきゃ、ちゃんとハードルを飛びこえなきゃって、苦しんで、、
だけど、もがいて頑張って良かったです。
 
 
そして、そんな私たちの気持ちを
わかってくれてたかのような、スタッフさんの優しさに感動しました。
 
本当に、有難うございました。
 
沢山の方たちに支えられて、感謝で胸がいっぱいになりました。
 
 
この舞台に携わってくださった、全ての方々
(もちろん、作り手だけでなくお客さんも全員！)
に、感謝ですっ( *ˆoˆ* )
 
 
他のメンバーも毎回舞台を観に来てくれて、改めてこのグループが好きだと思ったし、その一員でいれて幸せだと思いました。
 
 
演出の川尻さんはじめ、演出助手の西岡さん、振り付けの竹森さん、落語の桂枝太郎さん、音楽のあらいふとしさん、
他にもいっぱいいっぱい、色んな方にお世話になりました。
有難うございました
 
この出会いと経験は、これからの自分にとって、きっと大きなものになると思います。
