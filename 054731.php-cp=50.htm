2020/0125Sat井上小百合ヽ(｡･ω･｡)天国の本屋



ミュージカル『天国の本屋』
あっという間に初日があけて、
今日が千秋楽でした


バタバタしてて
ブログ更新できなくて
すみません！



ユイちゃん。

自分にかなり近い気がしてて、
お芝居してる実感がないくらいです。

昨年から、年をまたいでの
稽古だったので、色んなイベントが
盛りだくさんですごく楽しい稽古場でした

カンパニーの皆さんもすごく
あたたかくって、毎日が
笑顔が溢れる現場でした
感謝です


メンバーも、くぼてゃん、れなちさん、
まいちゅん、ゆなちゃん、かながわちゃん
みんな色々忙しい中ちゃんと
差し入れまで買ってきてくれて
有難さしかないし、
くぼてゃんとれなちさんはお手紙まで（ ; ; ）





ほんとに有難う
いい後輩たちよ（ ; ; ）（ ; ; ）



それから、やじさんも来てくれました！




うれしい〜( ˶ˆᴗˆ˶ )



残すは大阪サンケーブリーゼにて
3日間5公演！

東京公演中に髪の毛が
伸びてきてしまったので、
切りに行きたいなあ





では！


さゆ

「天国の本屋」東京千穐楽おめでとうございます。本当に素晴らしい作品になりましたね。大成功おめでとうございます。稽古期間から体調面で苦労されているのを聞いていたので気を揉んでいましたが、さゆ自身の努力と、そして周囲の支えのおかげで、公演では日々素晴らしいパフォーマンスを拝見することができました。こころから「ありがとう」と言いたいです。

さゆだけでなく、カンパニーのすべての皆さまにも「ありがとう」を。「暗闇に押し潰されそうになった時でも、そこにはいつでも皆がいた。大切な作品。一人でくじけそうになった時でも誰かがあたたかなこの劇場に帰してくれた。ありがとう。ありがとう」

それにしてもさゆの芝居、そして歌声は本当に素晴らしかった。どこまでも澄み渡っていて、美しく、でもどこか儚げな歌声が最後、戸惑いながらも喜びに染まっていくのが感動的でした。そして、ユイの涙と表情で会場全体がえも言われぬ雰囲気に包まれていました。劇場がまさに一つとなるお芝居の醍醐味を味わうことができました。
