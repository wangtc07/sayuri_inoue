2012/0926Wed井上小百合ノスタルジック。 ＋＋

 
 
 
だんだんと、秋めいてきましたね
 
暑さ寒さも彼岸までと言いますが、
今年は彼岸花、見れるかなあ(*^_^*)？
 
 
この時期になると、
毎年学校の通学路に彼岸花が咲いてね、
それを見ながら帰るのが好きでした ∧( 'Θ' )∧ 夕やけこやけ～♪
 
 
 
そんな私、お花が好きで、お家でも
お花を育てているのですが
前にブログに載せたお花のタネ...
 
最近、まったく状況を報告していなかったので
どうなっちゃったの？？もしや枯らしちゃった！？(´･-･`)
と、思っていた人もいると思うのですが...（笑）
 
ちゃんと、忙しい仕事の日でも毎日お水をあげて、
大事に育てていましたよっ( ^_^ )
 
 
 
 
 
...でも、一向に変化がなくて...（ ;  ; ）笑
 
 
もしやこれは、ただの「 草 」なんじゃないか？！！
って、心配になっていたんですが（笑）
 
 
最近、 やっと 蕾が
できてきました(  >  <  )！！！！
 
 
 
 
ほらっ
 
 
少しずつだけど、しっかりと確実に
成長しているようです(  *^_^*  )
 
 
よかったーっ
 
 
元気に育ってね＾＾ほっこり
 
 
 
 
 
そして、今日は せいたんと、とまとと
ご飯に行きました( =^-^= )
 
お休みをもらったけど、どうしていいかわからなくて
取り敢えず朝からずっと勉強をしてて、寂しくなって、
餓死しちゃうって連絡したら、2人が助けてくれました
 
 
久しぶりに、お腹いっぱい食べちゃって
お腹苦しかったけど幸せでした～
 
 
そのあとは、カラオケに行って消費消費っ( =^ω^= )！！
 
すごく楽しかったです(o^^o)
 
せいたん、とまと、有難う
 
 
実は、今日、せいたんが夢の中に出てきたんだよ～＼(^o^)／
 
でも、せいたんにもう別れようって言われる夢だった～＼(^o^)／
 
でも、そもそも付き合ってないよ～（泣)
 
なぜだ～～～～っ＼(^o^)／
 
まぁ、そのくらいせいたんのことが
好きってことなのかな(=^x^=)？せいたん好きよっ！えへ
 
ゴセイナイトさんごめん！
 
 
 
 
おやすみなさゆにゃん♪
 
 
 
 
 
さゆにゃんヽ(｡･ω･｡)
