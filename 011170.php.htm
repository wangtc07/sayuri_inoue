2013/0328Thu井上小百合 ヽ(｡･ω･｡) 神がうまれる！はるかちゃんという神が！そうさ！現実をぶち壊してがっちゃーん！バラバラ～

 
 
お早うございまーす！
 
 
カキカキ...
 
 
グリーンボーイ描いてみたよ(丿^ω^ヽ)えへへ
 
 
はるか姉さま完璧だ...可愛すぎる...！
 
たまこちゃんも可愛い...！たまこちゃんとコアラごっこしたいわたしもたまこちゃんのベットでうりゅりゅりゅ
 
最近家に帰ると、ずーっと録り溜めしたアニメと特撮をみています( `^ω^)=3
 
しあわせしあわせ！
 
寝る時間を忘れてくぎ付けになっている( `^ω^)=3
 
一回見たやつも、もいっかい見てみるとまた面白かったりして、止まらない！！
 
いや～、我ながらすごく人生を謳歌している気がする！ぐふぐふ( `^ω^)=3
 
ところでどっこいキョウリュウジャーが個人的にいろいろとツボなんですが！
ツボすぎるんだが！好きすぎるんだが！！！
 
キョウリュウチェンジ！
ファイヤー！！！
 
ガブリンチョォォオオ( `^ω^)=3
 
レンジャーレンジャーしてて好きだなー( *´ω`* )
ザ・スーパー戦隊って感じで！
よいっ！とてもよいっ＼( ^o^ )／！
敵もキャラがあって毎回 面白いし！
コレコレ！これが好きなんだよーぐふぐふぐふ！！って色々と、くすぐられるものがあるわ！
やっぱ好きだーーー！
ゴーバスターズは声優さんが素晴らしくて、私としてはたまらなかったから 
あれはあれで良かったけど！
 
今回は、いつもクールなブルーが真逆なキャラだから面白いねっ！( ^o^ )
あとグリーンボーイと、ブラックの絡み、好きだよ( *^p^* )
 
 
わたし、中学に入学してすぐの、クラスでの自己紹介で、一人づつ好きなものを言っていくというので、
特撮とアニメのこと言ったら、理解してもらえなくて（笑）
あれぇ...σ(^ω^;) ってなったニガイ思い出があるんです（笑）
 
 
が！しかーし！いまは、そういう女の子も増えて、本当にいい時代になりました...
 
うれしいっす( `^ω^)=3
 
でもちょっと悲しかったりもします。
おぬし、知らなかったじゃんかよーo(｀ω´ )oって思うことも。
 
 
でも、いまは、好きって堂々と言えるからうれしいなー。
中学のときは、「 しょ、小説かいてるよ(｀＾д´＾；)！ 」 って言って、
漫画を描いていたのはヒミツです。
 
 
カキカキ...
 
 
 
さあて、これは誰でしょ～( *^o^* )？
 
 
 
 
 
生写真でした、メイドさまコス...  
恥ずかしいです。
 
 
昨日の、PON!祭りきて下さった方
有難うございました！
寒くなかったですか？風邪は引いてないですか？
今日のZIP！春フェスも宜しくお願いしまーす( *^o^* )
東京ドームシティだからな！高まるぜっ！！！
キョウリュウジャーとぼくと握手☆してきたい！！！！
 
 
 
 
 
やっぱり恥ずかしいので、衣装の中に隠れる。
 
 
どれがわたしか、わかりますかー？
 
 
 
って、わかるか！（笑）
 
 
 
 
 
さゆにゃん
