2014/0722Tue井上小百合ヽ(。・ω ・。)夏 だ！

 
 
昨日は、幕張にて個別握手会でした＊＊
来て下さった皆さん、有難う*\(^o^)/*
 
 
今回は、前のブログで書いた通り
夏らしい服装をしました！
 
 
 
実は、こないだ家族でお祭りに行ってきたの♪
その時の浴衣を着たよっっ
 
お庭で妹に撮ってもらった写真 ^^*
 
 
 
 
姪っ子と甥っ子と！
かわいい(  //＾＾//  )
 
お祭りなんて何年ぶりだろう... 楽しかった。
 
 
 
 
では、各部ごとのレポート(  ^o^  )☆
 
 
１部 ＊  ふわふわサイドポニー
 
 
 
 
２部 ＊  くるんて丸めて両サイドの髪をちょこっと出したやつ
 
 
 
 
３部  ＊ ツインをくるんてしたやつ
陽菜ちゃんと(#^.^#)
 
 
 
 
４部は、髪型同じで浴衣をお着替え ＊
 
 
５部 ＊  (リクエスト)ハーフツイン
 
百合のお花柄です ＊
気付いてくれた方沢山いて嬉しかったです(* ^_^ *)
 
 
 
浴衣、、一人で着れたの♪
ちゃんと、自分で着付けできました！頑張った*\(^o^)/*
 
女子力( ・´ー・｀)ドヤ！ ......と、言いたいところだけど
 
なんと「ゲタを忘れる」という大失態。（笑）
 
 
ありゃ(´・д・；) ?  { まあサンダルも、それはそれで、なんかオシャレに見えなくもないから まあいっか！笑 )
 
 
ドジで天然でおっちょこちょいだけど、全く気にしないのが私（笑）
 
自慢ですはい！
 
 
そういえば、花火大会とか行ったことないなあ。行ってみたいなあ。
あの夜空に散るでっかい花火を
目の前で見られたら、キレイなんだろうなあ〜
 
 
 ー
 
質問にこたえるよ(ミ＾o＾ミ)
 
 
▼  Name :  仙台のまーくん さん
 
◎  京都みたいにのど自慢やるとしたら、さゆちゃんなら何歌う？
 
＊  うーん、aikoさんとか家入レオさんとかかなあ(o^^o)
 
 
 
▼  Name :  フジキセキ  さん
 
◎  もうすぐ夏だけど、さゆは海すきですか？
 
＊  埼玉県民だから、海には
憧れと、少し抵抗がある（笑）
 
 
 
▼  Name :  あべちゃん  さん
 
◎  さゆは夏休みの嫌な思い出なんかあったりする？？ 笑
 
＊  吹奏楽部の夏コンがあるから、
ずーっと部活で、遊んだりお祭り行ったり一切出来なかったし、練習きつかったー(´^ω^`;)
 
あと、習い事の合宿で、女子しかいない習い事だったから楽しかったんだけど...
隣の部屋の先輩が「怖い話しようよ!」って言ってきて、
 
怖いのがめちゃめちゃ苦手な私（当時小3）は、「せ、先輩...トイレに行ってきます!」って言って逃げたんだけど
逆に一人でトイレ行くほうが怖くて後悔。。
 
でも、戻ってきたら私の部屋の先輩が泣いてたから
 
一体、どんな恐ろしいことがおこったんだー:(；゛゜'ω゜'):
ってガクブルした思い出（笑）
 
 
 
 ー
 
 
休憩中に、きいちゃん、ろってぃー、ゆったんの生誕祭もありました！
 
お誕生日おめでとうo(^o^)o
 
 
 
そして、れなりんの卒業セレモニーもありました。
 
約三年間、お疲れさま。
それから、有難うね。
 
学業もお仕事も、大変なのによく頑張ってたね。
れなりんの、明るくて元気なとこ、時々ちょこっと抜けてること、
みんな大好きですo(^o^)o
 
れなりんと一番思い出が深いのは、4枚目。
ポジションも隣で、ユニット曲もペアになることが多かったから
もうれなりんと一緒に歌えなくなっちゃうの悲しい...
来年のバースデーライブはもうれなりんとのペアは無いんだなあ...
やっぱり、寂しいな...
 
でも、これからも、お互い夢に向かって頑張ろうね！
 
ほんとに三年間有難う。
 
 
 
では！！！
 
 
 
さゆヽ(。・ω・。)正義！
